# 25 - switch Statement in C++

## What is switch Statement?
The `switch` statement is a multi-way decision maker that tests a variable against a list of values. It's often used as an alternative to long `else-if` ladders when comparing a single variable against multiple constant values.

## Basic Syntax

```cpp
switch (expression) {
    case value1:
        // code executes if expression == value1
        break;
    case value2:
        // code executes if expression == value2
        break;
    case value3:
        // code executes if expression == value3
        break;
    default:
        // code executes if no case matches
}
```

## Simple Example

```cpp
#include <iostream>
using namespace std;

int main() {
    int day = 3;
    
    switch (day) {
        case 1:
            cout << "Monday" << endl;
            break;
        case 2:
            cout << "Tuesday" << endl;
            break;
        case 3:
            cout << "Wednesday" << endl;
            break;
        case 4:
            cout << "Thursday" << endl;
            break;
        case 5:
            cout << "Friday" << endl;
            break;
        default:
            cout << "Weekend" << endl;
    }
    
    return 0;
}
```
Output:
```
Wednesday
```

## How switch Works

```cpp
#include <iostream>
using namespace std;

int main() {
    int choice;
    
    cout << "Menu:" << endl;
    cout << "1. Start Game" << endl;
    cout << "2. Load Game" << endl;
    cout << "3. Settings" << endl;
    cout << "4. Exit" << endl;
    cout << "Choice: ";
    cin >> choice;
    
    switch (choice) {
        case 1:
            cout << "Starting game..." << endl;
            break;
        case 2:
            cout << "Loading game..." << endl;
            break;
        case 3:
            cout << "Opening settings..." << endl;
            break;
        case 4:
            cout << "Goodbye!" << endl;
            break;
        default:
            cout << "Invalid choice!" << endl;
    }
    
    return 0;
}
```

## The break Statement

`break` exits the switch block. Without it, execution "falls through" to the next case.

### Without break (Fall-through)
```cpp
#include <iostream>
using namespace std;

int main() {
    int num = 2;
    
    cout << "Without break:" << endl;
    switch (num) {
        case 1:
            cout << "Case 1" << endl;
        case 2:
            cout << "Case 2" << endl;  // This runs
        case 3:
            cout << "Case 3" << endl;  // This also runs!
        default:
            cout << "Default" << endl; // This also runs!
    }
    
    return 0;
}
```
Output:
```
Without break:
Case 2
Case 3
Default
```

### With break
```cpp
#include <iostream>
using namespace std;

int main() {
    int num = 2;
    
    cout << "With break:" << endl;
    switch (num) {
        case 1:
            cout << "Case 1" << endl;
            break;
        case 2:
            cout << "Case 2" << endl;  // This runs
            break;                      // Then exits
        case 3:
            cout << "Case 3" << endl;
            break;
        default:
            cout << "Default" << endl;
    }
    
    return 0;
}
```
Output:
```
With break:
Case 2
```

## Using Fall-through Intentionally

```cpp
#include <iostream>
using namespace std;

int main() {
    char grade = 'B';
    
    switch (grade) {
        case 'A':
            cout << "Excellent! ";
            // Fall through to also show "Passed"
        case 'B':
        case 'C':
        case 'D':
            cout << "Passed";
            break;
        case 'F':
            cout << "Failed";
            break;
        default:
            cout << "Invalid grade";
    }
    
    return 0;
}
```
Output:
```
Passed
```

## switch with char

```cpp
#include <iostream>
using namespace std;

int main() {
    char operation;
    double a, b;
    
    cout << "Enter operation (+, -, *, /): ";
    cin >> operation;
    cout << "Enter two numbers: ";
    cin >> a >> b;
    
    switch (operation) {
        case '+':
            cout << a << " + " << b << " = " << a + b << endl;
            break;
        case '-':
            cout << a << " - " << b << " = " << a - b << endl;
            break;
        case '*':
            cout << a << " * " << b << " = " << a * b << endl;
            break;
        case '/':
            if (b != 0) {
                cout << a << " / " << b << " = " << a / b << endl;
            } else {
                cout << "Cannot divide by zero!" << endl;
            }
            break;
        default:
            cout << "Invalid operation!" << endl;
    }
    
    return 0;
}
```

### Example 2: Simple Calculator Menu
```cpp
#include <iostream>
using namespace std;

int main() {
    int choice;
    double num1, num2;
    
    do {
        cout << "\n=== Calculator Menu ===" << endl;
        cout << "1. Add" << endl;
        cout << "2. Subtract" << endl;
        cout << "3. Multiply" << endl;
        cout << "4. Divide" << endl;
        cout << "5. Exit" << endl;
        cout << "Choice: ";
        cin >> choice;
        
        if (choice >= 1 && choice <= 4) {
            cout << "Enter two numbers: ";
            cin >> num1 >> num2;
        }
        
        switch (choice) {
            case 1:
                cout << "Result: " << num1 + num2 << endl;
                break;
            case 2:
                cout << "Result: " << num1 - num2 << endl;
                break;
            case 3:
                cout << "Result: " << num1 * num2 << endl;
                break;
            case 4:
                if (num2 != 0) {
                    cout << "Result: " << num1 / num2 << endl;
                } else {
                    cout << "Cannot divide by zero!" << endl;
                }
                break;
            case 5:
                cout << "Goodbye!" << endl;
                break;
            default:
                cout << "Invalid choice!" << endl;
        }
    } while (choice != 5);
    
    return 0;
}
```

## switch vs else-if

```cpp
#include <iostream>
using namespace std;

int main() {
    int choice = 3;
    
    // Using else-if
    cout << "Else-if version:" << endl;
    if (choice == 1) {
        cout << "Option 1" << endl;
    } else if (choice == 2) {
        cout << "Option 2" << endl;
    } else if (choice == 3) {
        cout << "Option 3" << endl;  // This runs
    } else if (choice == 4) {
        cout << "Option 4" << endl;
    } else {
        cout << "Default" << endl;
    }
    
    // Using switch (cleaner for multiple values)
    cout << "\nSwitch version:" << endl;
    switch (choice) {
        case 1:
            cout << "Option 1" << endl;
            break;
        case 2:
            cout << "Option 2" << endl;
            break;
        case 3:
            cout << "Option 3" << endl;  // This runs
            break;
        case 4:
            cout << "Option 4" << endl;
            break;
        default:
            cout << "Default" << endl;
    }
    
    return 0;
}
```

## What switch Can and Cannot Do

### ✅ Allowed
```cpp
int num = 5;
char ch = 'A';
enum Day { MON, TUE, WED };

switch (num) { }     // ✅ integers
switch (ch) { }      // ✅ characters
switch (MON) { }     // ✅ enums
```

### ❌ Not Allowed
```cpp
double d = 3.14;
string s = "hello";

// switch (d) { }     // ❌ No floating point
// switch (s) { }     // ❌ No strings
// switch (x > 5) { } // ❌ No expressions
```

## Common Mistakes

### Mistake 1: Missing break
```cpp
int x = 2;

// Wrong - falls through
switch (x) {
    case 1:
        cout << "One" << endl;
    case 2:
        cout << "Two" << endl;  // This runs
    case 3:
        cout << "Three" << endl; // This also runs!
}

// Correct
switch (x) {
    case 1:
        cout << "One" << endl;
        break;
    case 2:
        cout << "Two" << endl;
        break;
    case 3:
        cout << "Three" << endl;
        break;
}
```

### Mistake 2: Duplicate Cases
```cpp
int x = 5;

// Wrong
switch (x) {
    case 1:
        cout << "One" << endl;
        break;
    case 1:  // ❌ Duplicate case
        cout << "Also one?" << endl;
        break;
}

// Correct
switch (x) {
    case 1:
        cout << "One" << endl;
        break;
    // No duplicate
}
```

### Mistake 3: Forgetting default
```cpp
int x = 100;

// Wrong - no handling for unexpected values
switch (x) {
    case 1:
        cout << "One" << endl;
        break;
    case 2:
        cout << "Two" << endl;
        break;
}
// What if x is 100? Nothing happens!

// Correct
switch (x) {
    case 1:
        cout << "One" << endl;
        break;
    case 2:
        cout << "Two" << endl;
        break;
    default:
        cout << "Unknown value: " << x << endl;
}
```

### Mistake 4: Using Variables in Cases
```cpp
int a = 5;
int b = 10;

// Wrong
switch (a) {
    case b:  // ❌ Case must be constant
        cout << "Equal" << endl;
        break;
}

// Correct
const int VALUE = 10;
switch (a) {
    case VALUE:  // ✅ Constant
        cout << "Equal" << endl;
        break;
}
```

### Mistake 5: Complex Conditions
```cpp
int x = 5;

// Wrong - switch can't handle complex conditions
switch (x) {
    case x > 0:  // ❌ Not allowed
        cout << "Positive" << endl;
        break;
}

// Use if-else instead
if (x > 0) {
    cout << "Positive" << endl;
}
```

## Multiple Cases Same Code

## Nested switch Statements

```cpp
#include <iostream>
using namespace std;

int main() {
    int category, item;
    
    cout << "Category (1-2): ";
    cin >> category;
    
    switch (category) {
        case 1:
            cout << "Electronics" << endl;
            cout << "Item (1-3): ";
            cin >> item;
            
            switch (item) {
                case 1:
                    cout << "Laptop" << endl;
                    break;
                case 2:
                    cout << "Phone" << endl;
                    break;
                case 3:
                    cout << "Tablet" << endl;
                    break;
                default:
                    cout << "Invalid item" << endl;
            }
            break;
            
        case 2:
            cout << "Clothing" << endl;
            cout << "Item (1-3): ";
            cin >> item;
            
            switch (item) {
                case 1:
                    cout << "Shirt" << endl;
                    break;
                case 2:
                    cout << "Pants" << endl;
                    break;
                case 3:
                    cout << "Jacket" << endl;
                    break;
                default:
                    cout << "Invalid item" << endl;
            }
            break;
            
        default:
            cout << "Invalid category" << endl;
    }
    
    return 0;
}
```

## Quick Reference

| Component | Purpose |
|-----------|---------|
| `switch(expr)` | Expression to test (integer, char, enum) |
| `case value:` | Compare expression to constant value |
| `break;` | Exit switch (optional but important) |
| `default:` | Runs when no case matches (optional) |

## When to Use switch vs if-else

| Use switch when: | Use if-else when: |
|-----------------|-------------------|
| Testing single variable | Testing complex conditions |
| Many discrete values | Ranges of values |
| Values are constants | Conditions with && or \|\| |
| Cleaner than long else-if | Floating point comparisons |
| Enums or characters | String comparisons |

## Summary

- **switch** tests a variable against multiple constant values
- Works with integers, characters, and enums
- **break** prevents fall-through to next case
- **default** handles unmatched cases
- Multiple cases can share same code
- Cases must be constant expressions
- More readable than long else-if ladders
- Cannot use with strings, floats, or expressions
- Fall-through can be useful intentionally
- Always include default for robustness

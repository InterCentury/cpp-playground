# 05 - cin for Input in C++

## What is cin?
`cin` (pronounced "see-in") is used to take input from the user. It stands for "character input".

## Basic Syntax
```cpp
cin >> variable;
```
- `>>` is the extraction operator (takes input from keyboard and stores in variable)

## Simple Examples

### Example 1: Taking a Number
```cpp
#include <iostream>
using namespace std;

int main() {
    int age;
    
    cout << "Enter your age: ";
    cin >> age;
    
    cout << "You are " << age << " years old";
    return 0;
}
```
```
Enter your age: 25
You are 25 years old
```

### Example 2: Taking Text
```cpp
string name;
cout << "Enter your name: ";
cin >> name;
cout << "Hello " << name;
```
```
Enter your name: John
Hello John
```

## Taking Multiple Inputs

### Method 1: Separate cin statements
```cpp
int x, y;
cout << "Enter first number: ";
cin >> x;
cout << "Enter second number: ";
cin >> y;
cout << "Sum = " << x + y;
```

### Method 2: Chain with >>
```cpp
int x, y;
cout << "Enter two numbers: ";
cin >> x >> y;
cout << "Sum = " << x + y;
```
```
Enter two numbers: 10 20
Sum = 30
```

### Method 3: Mix different types
```cpp
string name;
int age;
cout << "Enter name and age: ";
cin >> name >> age;
cout << name << " is " << age << " years old";
```
```
Enter name and age: Alex 25
Alex is 25 years old
```

## Problem with cin and Strings

### cin stops at spaces
```cpp
string name;
cout << "Enter full name: ";
cin >> name;           // Only reads first word
cout << name;
```
```
Enter full name: John Smith
John
```

### Solution: Use getline()
```cpp
string fullName;
cout << "Enter full name: ";
getline(cin, fullName);
cout << fullName;
```
```
Enter full name: John Smith
John Smith
```

### cin.ignore() problem with getline
```cpp
int age;
string name;

cout << "Enter age: ";
cin >> age;                    // Leaves \n in buffer

cout << "Enter name: ";
getline(cin, name);            // Reads the leftover \n
cout << name;                   // Prints nothing!
```

### Fix with cin.ignore()
```cpp
int age;
string name;

cout << "Enter age: ";
cin >> age;
cin.ignore();                   // Clear the buffer

cout << "Enter name: ";
getline(cin, name);
cout << "Age: " << age << ", Name: " << name;
```
```
Enter age: 25
Enter name: John Smith
Age: 25, Name: John Smith
```

## Input Validation Examples

### Check if input is valid
```cpp
int number;
cout << "Enter a number: ";

if (cin >> number) {
    cout << "You entered: " << number;
} else {
    cout << "Invalid input!";
    cin.clear();              // Clear error flag
    cin.ignore(1000, '\n');   // Clear buffer
}
```

### Keep asking until valid input
```cpp
int age;
cout << "Enter age: ";

while (!(cin >> age)) {
    cout << "Invalid! Enter a number: ";
    cin.clear();
    cin.ignore(1000, '\n');
}
cout << "Age: " << age;
```

## Quick Reference Table

| What to Input | How to Write | Example |
|--------------|--------------|---------|
| Single number | `cin >> var;` | `cin >> x;` |
| Multiple numbers | `cin >> a >> b;` | `cin >> x >> y;` |
| Single word | `cin >> str;` | `cin >> name;` |
| Full line | `getline(cin, str);` | `getline(cin, address);` |
| Character | `cin >> ch;` | `cin >> grade;` |
| After cin >> | `cin.ignore();` then `getline()` | Clear buffer first |

## Common Errors

### Error 1: Missing &
```cpp
// Wrong (C style - not C++)
scanf("%d", age);

// Correct
cin >> age;
```

### Error 2: Wrong operator
```cpp
// Wrong
cin << age;

// Correct
cin >> age;
```

### Error 3: No variable
```cpp
// Wrong
cin >> ;

// Correct
cin >> age;
```

### Error 4: Wrong data type
```cpp
int num;
cin >> num;  // User enters "hello" -> num becomes 0 or error
```

### Error 5: Forgetting & with getline
```cpp
// Wrong
getline(cin, name);  // Actually correct in C++!
// Wait - this is actually correct for strings
// No & needed with C++ strings
```

## cin vs getline

| Feature | cin >> | getline(cin, str) |
|---------|--------|-------------------|
| Reads | One word | Entire line |
| Stops at | Space, tab, enter | Enter only |
| Skips whitespace | Yes | No |
| Buffer issues | Leaves \n | Reads \n as line end |
| Use for | Single words, numbers | Full sentences, addresses |

## Important Tips

### Tip 1: Always prompt user
```cpp
// Bad
cin >> age;

// Good
cout << "Enter age: ";
cin >> age;
```

### Tip 2: Use meaningful variable names
```cpp
// Bad
cin >> a;

// Good
cin >> studentAge;
```

### Tip 3: Clear buffer when switching
```cpp
cin >> number;
cin.ignore();        // Clear before getline
getline(cin, text);
```

### Tip 4: Validate when needed
```cpp
if (cin >> num) {
    // Use num
} else {
    // Handle error
}
```

## Summary

- `cin >> variable` takes input from keyboard
- Use `>>` extraction operator (not `<<`)
- Multiple inputs can be chained: `cin >> a >> b`
- `cin` stops at spaces for strings
- Use `getline(cin, str)` for full lines
- `cin.ignore()` clears the input buffer
- Always prompt user before taking input
- Different data types work with same `cin >>`
- Check for invalid input when needed
- Every cin statement ends with `;`

## Basic Template with Input
```cpp
#include <iostream>

int main() {

    int num;
    string word;
    
    cout << "Enter number: ";

    cout << "Enter word: ";

    // For full lines
```

# 21 - if Statement in C++

## What is an if Statement?
The `if` statement is used to execute a block of code only if a specified condition is true. It's the most basic form of decision-making in programming.

## Basic Syntax

```cpp
if (condition) {
    // code to execute if condition is true
}
```

## Simple Examples

```cpp
#include <iostream>
using namespace std;

int main() {
    int age = 20;
    
    if (age >= 18) {
        cout << "You are an adult." << endl;
    }
    
    cout << "Program continues..." << endl;
    
    return 0;
}
```
Output:
```
You are an adult.
Program continues...
```

## How if Statement Works

```cpp
#include <iostream>
using namespace std;

int main() {
    int number;
    
    cout << "Enter a number: ";
    cin >> number;
    
    if (number > 0) {
        cout << "The number is positive." << endl;
    }
    
    if (number < 0) {
        cout << "The number is negative." << endl;
    }
    
    if (number == 0) {
        cout << "The number is zero." << endl;
    }
    
    return 0;
}
```

## if with Different Conditions

### Using Comparison Operators
```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 10, y = 5;
    
    if (x > y) {
        cout << "x is greater than y" << endl;
    }
    
    if (x == y) {
        cout << "x equals y" << endl;
    }
    
    if (x != y) {
        cout << "x is not equal to y" << endl;
    }
    
    return 0;
}
```

### Using Logical Operators
```cpp
#include <iostream>
using namespace std;

int main() {
    int age = 25;
    bool hasLicense = true;
    
    // AND (&&) - both must be true
    if (age >= 18 && hasLicense) {
        cout << "You can drive!" << endl;
    }
    
    // OR (||) - at least one must be true
    bool isWeekend = true;
    bool isHoliday = false;
    
    if (isWeekend || isHoliday) {
        cout << "You can sleep in!" << endl;
    }
    
    // NOT (!) - reverse the condition
    bool isRaining = false;
    
    if (!isRaining) {
        cout << "You can go outside!" << endl;
    }
    
    return 0;
}
```

## if with Boolean Variables

```cpp
#include <iostream>
using namespace std;

int main() {
    bool isStudent = true;
    bool hasDiscount = false;
    
    // Directly use boolean variable
    if (isStudent) {
        cout << "Student discount available!" << endl;
    }
    
    // Using NOT operator
    if (!hasDiscount) {
        cout << "No discount applied." << endl;
    }
    
    // Combining booleans
    if (isStudent && !hasDiscount) {
        cout << "Eligible for new student discount!" << endl;
    }
    
    return 0;
}
```

## Nested if Statements

```cpp
#include <iostream>
using namespace std;

int main() {
    int num = 15;
    
    if (num > 0) {
        cout << "Number is positive." << endl;
        
        if (num % 2 == 0) {
            cout << "Number is even." << endl;
        } else {
            cout << "Number is odd." << endl;
        }
        
        if (num > 10) {
            cout << "Number is greater than 10." << endl;
        }
    }
    
    return 0;
}
```

## Common Mistakes

### Mistake 1: Using = Instead of ==
```cpp
int x = 5;

// Wrong - assignment, not comparison
if (x = 10) {
    cout << "This always runs!" << endl;  // x becomes 10
}

// Correct
if (x == 10) {
    cout << "This runs only if x equals 10" << endl;
}
```

### Mistake 2: Missing Braces
```cpp
int score = 75;

// Wrong - only first line is in if
if (score >= 60)
    cout << "You passed!" << endl;
    cout << "Congratulations!" << endl;  // This runs always!

// Correct
if (score >= 60) {
    cout << "You passed!" << endl;
    cout << "Congratulations!" << endl;
}
```

### Mistake 3: Semicolon After if
```cpp
int age = 20;

// Wrong - semicolon ends the if statement
if (age >= 18); {
    cout << "Adult" << endl;  // Always runs!
}

// Correct
if (age >= 18) {
    cout << "Adult" << endl;
}
```

### Mistake 4: Wrong Condition Logic
```cpp
int x = 5;

// Wrong - doesn't work as expected
if (1 < x < 10) {  // (1 < x) < 10 = true < 10 = true
    cout << "x is between 1 and 10" << endl;  // Always runs!
}

// Correct
if (x > 1 && x < 10) {
    cout << "x is between 1 and 10" << endl;
}
```

### Mistake 5: Forgetting to Convert
```cpp
char grade = 'A';

// Wrong - comparing char with string
if (grade == "A") {  // "A" is a string, 'A' is a char
    cout << "Excellent!" << endl;
}

// Correct
if (grade == 'A') {
    cout << "Excellent!" << endl;
}
```

## Multiple Conditions with && and ||

```cpp
#include <iostream>
using namespace std;

int main() {
    int age = 25;
    double income = 50000;
    bool hasJob = true;
    bool goodCredit = true;
    
    // All conditions must be true
    if (age >= 18 && age <= 65 && hasJob && income > 30000) {
        cout << "Eligible for loan - meets all criteria" << endl;
    }
    
    // At least one condition must be true
    if (age > 60 || income < 20000 || !hasJob) {
        cout << "Consider special assistance programs" << endl;
    }
    
    // Complex combination
    if ((age >= 18 && hasJob) || (income > 100000 && goodCredit)) {
        cout << "Approved for credit card" << endl;
    }
    
    return 0;
}
```

## if Statement Style Guide

### Good Style
```cpp
// Always use braces
if (condition) {
    statement;
}

// Indent properly
if (condition) {
    statement1;
    statement2;
}

// Complex conditions on multiple lines
if ((age >= 18 && hasLicense) || 
    (age >= 16 && isStudent && hasPermit)) {
    cout << "Can drive" << endl;
}
```

### Bad Style
```cpp
// No braces - risky
if (condition) statement;

// No indentation
if (condition) {
statement;
}

// Too crowded
if(age>=18&&hasLicense){cout<<"Can drive"<<endl;}
```

## Quick Reference

| Syntax | Meaning |
|--------|---------|
| `if (condition)` | If condition is true |
| `if (x > 5)` | If x is greater than 5 |
| `if (x == 10)` | If x equals 10 |
| `if (isValid)` | If isValid is true |
| `if (!isValid)` | If isValid is false |
| `if (x && y)` | If both x and y are true |
| `if (x || y)` | If either x or y is true |

## Summary

- **if statement** executes code only when condition is true
- Condition must be in parentheses `()`
- Code block must be in braces `{}` (best practice)
- Use `==` for comparison, not `=`
- Multiple statements need braces
- Conditions can use comparison and logical operators
- Boolean variables can be used directly
- Nested ifs allow checking multiple conditions
- Always indent code properly for readability
- Test both true and false cases

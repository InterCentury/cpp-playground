# 11 - Boolean Type (bool) in C++

## What is bool?
`bool` is a data type that can hold only two values: `true` or `false`. It's used for logical operations and conditions.

## Basic Syntax

```cpp
bool isReady = true;
bool isFinished = false;
```

## Simple Examples

```cpp
#include <iostream>
using namespace std;

int main() {
    bool flag1 = true;
    bool flag2 = false;
    
    cout << "flag1: " << flag1 << endl;  // prints 1
    cout << "flag2: " << flag2 << endl;  // prints 0
    
    return 0;
}
```
Output:
```
flag1: 1
flag2: 0
```

## Boolean Output Formatting

```cpp
#include <iostream>
using namespace std;

int main() {
    bool a = true;
    bool b = false;
    
    cout << "Default output:" << endl;
    cout << "a: " << a << endl;  // 1
    cout << "b: " << b << endl;  // 0
    
    cout << "\nWith boolalpha:" << endl;
    cout << boolalpha;  // Show true/false instead of 1/0
    cout << "a: " << a << endl;  // true
    cout << "b: " << b << endl;  // false
    
    cout << "\nBack to default:" << endl;
    cout << noboolalpha;
    cout << "a: " << a << endl;  // 1
    
    return 0;
}
```

## Declaring and Initializing bool

### Different Ways
```cpp
#include <iostream>
using namespace std;

int main() {
    // Direct initialization
    bool b1 = true;
    bool b2 = false;
    
    // Constructor style
    bool b3(true);
    bool b4(false);
    
    // Uniform initialization (C++11)
    bool b5{true};
    bool b6{false};
    
    // Zero initialization
    bool b7 = {};  // false
    bool b8{};
    
    cout << boolalpha;
    cout << "b1: " << b1 << endl;  // true
    cout << "b2: " << b2 << endl;  // false
    cout << "b7: " << b7 << endl;  // false
    
    return 0;
}
```

## Boolean Conversion Rules

Any value can be converted to bool:
- **0** → `false`
- **Non-zero** → `true`

```cpp
#include <iostream>
using namespace std;

int main() {
    bool b1 = 0;      // false
    bool b2 = 1;      // true
    bool b3 = 42;     // true (any non-zero)
    bool b4 = -5;     // true
    bool b5 = 0.0;    // false
    bool b6 = 3.14;   // true
    bool b7 = 'A';    // true (non-zero ASCII)
    bool b8 = '\0';   // false (null character)
    
    cout << boolalpha;
    cout << "b1 (0): " << b1 << endl;
    cout << "b2 (1): " << b2 << endl;
    cout << "b3 (42): " << b3 << endl;
    cout << "b4 (-5): " << b4 << endl;
    cout << "b5 (0.0): " << b5 << endl;
    cout << "b6 (3.14): " << b6 << endl;
    
    return 0;
}
```

## Boolean with Logical Operators

```cpp
#include <iostream>
using namespace std;

int main() {
    bool a = true;
    bool b = false;
    
    cout << boolalpha;
    cout << "a = " << a << ", b = " << b << endl;
    
    // Logical operators
    cout << "a && b: " << (a && b) << endl;  // false (AND)
    cout << "a || b: " << (a || b) << endl;  // true  (OR)
    cout << "!a: " << (!a) << endl;           // false (NOT)
    cout << "!b: " << (!b) << endl;           // true
    
    return 0;
}
```

## Boolean with Comparison Operators

```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 10, y = 5;
    
    bool result1 = (x > y);    // true
    bool result2 = (x < y);    // false
    bool result3 = (x == y);   // false
    bool result4 = (x != y);   // true
    bool result5 = (x >= 10);  // true
    bool result6 = (y <= 3);   // false
    
    cout << boolalpha;
    cout << "10 > 5  : " << result1 << endl;
    cout << "10 < 5  : " << result2 << endl;
    cout << "10 == 5 : " << result3 << endl;
    cout << "10 != 5 : " << result4 << endl;
    
    return 0;
}
```

## Common Mistakes

### Mistake 1: Assignment vs Comparison
```cpp
// Wrong - assignment instead of comparison
if (x = 5) { }  // Always true, x becomes 5

// Correct
if (x == 5) { }
```

### Mistake 2: Extra Semicolon
```cpp
// Wrong - semicolon ends the if
if (x > 5); {
    cout << "x is greater";  // Always runs!
}

// Correct
if (x > 5) {
    cout << "x is greater";
}
```

### Mistake 3: Overcomplicating
```cpp
// Wrong - redundant
if (isValid == true) { }

// Correct
if (isValid) { }

// Wrong
if (isValid == false) { }

// Correct
if (!isValid) { }
```

### Mistake 4: Using int for boolean
```cpp
// Wrong - C-style
int flag = 1;

// Correct
bool flag = true;
```

### Mistake 5: Forgetting boolalpha
```cpp
bool flag = true;
cout << flag;  // Prints 1, not "true"

// To print "true"/"false"
cout << boolalpha << flag;
```

## Boolean Best Practices

### ✅ Do This
```cpp
// Use meaningful names
bool isStudent = true;
bool hasLicense = false;
bool canVote = (age >= 18);

// Use in conditions directly
if (isReady) { }

// Use logical operators clearly
bool isValid = (username != "") && (password.length() >= 6);

// Return boolean from functions
bool isEven(int n) { return n % 2 == 0; }

// Use bool for flags
bool quit = false;
while (!quit) { }
```

### ❌ Avoid This
```cpp
// Don't compare with true/false
if (isValid == true) { }  // Redundant

// Don't use magic numbers
int flag = 1;  // Use bool

// Don't overcomplicate
if (x > 5 ? true : false)  // Just use x > 5

// Don't use int for boolean logic
int result = a && b;  // Use bool
```

## Quick Reference Table

| Operation | Code | Result |
|-----------|------|--------|
| Declare true | `bool b = true;` | true |
| Declare false | `bool b = false;` | false |
| AND | `a && b` | true if both true |
| OR | `a || b` | true if either true |
| NOT | `!a` | opposite |
| Convert 0 | `bool(0)` | false |
| Convert non-zero | `bool(42)` | true |
| Print as text | `cout << boolalpha` | shows true/false |
| Toggle | `flag = !flag;` | flips value |

## Boolean Truth Table

| a | b | a && b | a || b | !a |
|---|---|--------|--------|-----|
| false | false | false | false | true |
| false | true | false | true | true |
| true | false | false | true | false |
| true | true | true | true | false |

## Summary

- **bool** stores only `true` or `false`
- Default output shows `1` (true) or `0` (false)
- Use `boolalpha` to show `true`/`false`
- Any non-zero value converts to `true`
- Zero converts to `false`
- Use logical operators: `&&` (AND), `||` (OR), `!` (NOT)
- Comparison operators return bool
- Use meaningful names like `isReady`, `hasValue`
- Never compare with `== true` or `== false`
- Perfect for flags, conditions, and toggles
- Use in loops for control: `while (!quit)`

# 31 - break Statement in C++

## What is break Statement?
The `break` statement is used to exit a loop or switch statement immediately. When encountered, it terminates the nearest enclosing loop or switch and transfers control to the statement following it.

## Basic Syntax

```cpp
break;
```

## Simple Example

```cpp
#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 10; i++) {
        if (i == 5) {
            cout << "Break at i = " << i << endl;
            break;  // Exit loop when i is 5
        }
        cout << "i = " << i << endl;
    }
    
    cout << "Loop ended!" << endl;
    
    return 0;
}
```
Output:
```
i = 1
i = 2
i = 3
i = 4
Break at i = 5
Loop ended!
```

## break in Different Loops

### break in for Loop
```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "break in for loop:" << endl;
    for (int i = 1; i <= 10; i++) {
        if (i > 5) {
            break;
        }
        cout << i << " ";
    }
    cout << "\nLoop ended" << endl;
    
    return 0;
}
```

### break in while Loop
```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "break in while loop:" << endl;
    int i = 1;
    
    while (i <= 10) {
        if (i == 6) {
            break;
        }
        cout << i << " ";
        i++;
    }
    cout << "\nLoop ended" << endl;
    
    return 0;
}
```

### break in do-while Loop
```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "break in do-while loop:" << endl;
    int i = 1;
    
    do {
        if (i == 4) {
            break;
        }
        cout << i << " ";
        i++;
    } while (i <= 10);
    cout << "\nLoop ended" << endl;
    
    return 0;
}
```

## break in switch Statement

```cpp
#include <iostream>
using namespace std;

int main() {
    int choice;
    
    cout << "Enter choice (1-3): ";
    cin >> choice;
    
    switch (choice) {
        case 1:
            cout << "Option 1 selected" << endl;
            break;  // Without break, falls through to case 2
        case 2:
            cout << "Option 2 selected" << endl;
            break;
        case 3:
            cout << "Option 3 selected" << endl;
            break;
        default:
            cout << "Invalid choice" << endl;
    }
    
    return 0;
}
```

## Practical Examples

### Example 1: Search in Array
```cpp
#include <iostream>
using namespace std;

int main() {
    int numbers[] = {23, 45, 12, 67, 89, 34, 56};
    int size = sizeof(numbers) / sizeof(numbers[0]);
    int searchFor;
    bool found = false;
    
    cout << "Enter number to search: ";
    cin >> searchFor;
    
    for (int i = 0; i < size; i++) {
        if (numbers[i] == searchFor) {
            cout << "Found " << searchFor << " at index " << i << endl;
            found = true;
            break;  // Exit loop once found
        }
    }
    
    if (!found) {
        cout << searchFor << " not found in array" << endl;
    }
    
    return 0;
}
```

### Example 2: Prime Number Checker
```cpp
#include <iostream>
using namespace std;

int main() {
    int num;
    bool isPrime = true;
    
    cout << "Enter a number: ";
    cin >> num;
    
    if (num <= 1) {
        isPrime = false;
    } else {
        for (int i = 2; i <= num / 2; i++) {
            if (num % i == 0) {
                isPrime = false;
                break;  // Found divisor, not prime
            }
        }
    }
    
    if (isPrime) {
        cout << num << " is prime" << endl;
    } else {
        cout << num << " is not prime" << endl;
    }
    
    return 0;
}
```

```cpp
#include <iostream>
```

### Example 4: Early Exit on Error
```cpp
#include <iostream>
using namespace std;

int main() {
    int numbers[5];
    
    cout << "Enter 5 positive numbers:" << endl;
    
    for (int i = 0; i < 5; i++) {
        cout << "Number " << i + 1 << ": ";
        cin >> numbers[i];
        
        if (numbers[i] <= 0) {
            cout << "Error: Only positive numbers allowed!" << endl;
            cout << "Exiting input early." << endl;
            break;  // Exit loop on error
        }
    }
    
    cout << "\nNumbers entered: ";
    for (int i = 0; i < 5 && numbers[i] > 0; i++) {
        cout << numbers[i] << " ";
    }
    cout << endl;
    
    return 0;
}
```

### Example 5: Find First Negative Number
```cpp
#include <iostream>
using namespace std;

int main() {
```

## break in Nested Loops

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Break in nested loops:" << endl;
    
    for (int i = 1; i <= 3; i++) {
        cout << "Outer i = " << i << ": ";
        
        for (int j = 1; j <= 5; j++) {
            if (j == 3) {
                cout << "break at j=3 | ";
                break;  // Only breaks inner loop
            }
            cout << j << " ";
        }
        
        cout << endl;
    }
    
    return 0;
}
```
Output:
```
Outer i = 1: 1 2 break at j=3 | 
Outer i = 2: 1 2 break at j=3 | 
Outer i = 3: 1 2 break at j=3 | 
```

### Breaking Outer Loop (using flag)
```cpp
#include <iostream>
using namespace std;

int main() {
    bool found = false;
    
    for (int i = 1; i <= 3 && !found; i++) {
        for (int j = 1; j <= 3; j++) {
            cout << "(" << i << "," << j << ") ";
            
            if (i == 2 && j == 2) {
                cout << "Found! ";
                found = true;
                break;  // Breaks inner loop
            }
        }
        cout << endl;
        // Outer loop condition (!found) will stop next iteration
    }
    
    return 0;
}
```

### Breaking Both Loops (using goto)
```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Breaking both loops with goto:" << endl;
    
    for (int i = 1; i <= 3; i++) {
        for (int j = 1; j <= 3; j++) {
            cout << "(" << i << "," << j << ") ";
            
            if (i == 2 && j == 2) {
                cout << "Found! Breaking all loops." << endl;
                goto end;  // Jumps to label
            }
        }
        cout << endl;
    }
    
end:
    cout << "All loops ended" << endl;
    
    return 0;
}
```

## break in Infinite Loops

```cpp
#include <iostream>
using namespace std;

int main() {
    int count = 0;
    
    cout << "Infinite loop with break:" << endl;
    
    while (true) {
        count++;
        cout << "Iteration " << count << endl;
        
        if (count >= 5) {
            cout << "Break condition met!" << endl;
            break;  // Exit infinite loop
        }
    }
    
    cout << "Loop ended after " << count << " iterations" << endl;
    
    return 0;
}
```

## Common Mistakes

### Mistake 1: Break Outside Loop or Switch
```cpp
// Wrong
if (x > 5) {
    break;  // Error! break not in loop or switch
}

// Correct
while (condition) {
    if (x > 5) {
        break;  // OK
    }
}
```

### Mistake 2: Forgetting Break in switch
```cpp
int x = 2;

// Wrong - missing break causes fall-through
switch (x) {
    case 1:
        cout << "One" << endl;
    case 2:
        cout << "Two" << endl;  // This runs
    case 3:
        cout << "Three" << endl;  // This also runs!
}

// Correct
switch (x) {
    case 1:
        cout << "One" << endl;
        break;
    case 2:
        cout << "Two" << endl;
        break;
    case 3:
        cout << "Three" << endl;
        break;
}
```

### Mistake 3: Break in if Inside Loop (Works, but...)
```cpp
// This is fine, but make sure it's what you want
for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break;  // Exits loop when i=5
    }
    cout << i << " ";
}
// Output: 0 1 2 3 4
```

### Mistake 4: Thinking Break Exits All Nested Loops
```cpp
// Wrong assumption
for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        if (i == 1 && j == 1) {
            break;  // Only breaks inner loop
        }
        cout << i << "," << j << " ";
    }
    cout << endl;  // This still runs!
}
```

### Mistake 5: Unreachable Code After Break
```cpp
for (int i = 0; i < 10; i++) {
    break;
    cout << "This never prints!" << endl;  // Unreachable
}
```

## break vs continue vs return

| Statement | Effect |
|-----------|--------|
| `break` | Exits current loop or switch |
| `continue` | Skips to next iteration |
| `return` | Exits entire function |

```cpp
#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 5; i++) {
        if (i == 2) {
            continue;  // Skip printing 2
        }
        if (i == 4) {
            break;     // Exit loop at 4
        }
        cout << i << " ";
    }
    cout << "\nLoop ended" << endl;
    
    return 0;  // Exit function
}
```
Output:
```
1 3 
Loop ended
```

## When to Use break

### ✅ Good Uses of break
```cpp
// Found what you're looking for
for (int i = 0; i < size; i++) {
    if (array[i] == target) {
        found = true;
        break;
    }
}

// Error condition
while (validInput) {
    cin >> value;
    if (value < 0) {
        cout << "Invalid input!" << endl;
        break;
    }
    process(value);
}

// Menu exit
while (true) {
    displayMenu();
    cin >> choice;
    if (choice == 4) break;
    handleChoice(choice);
}
```

### ❌ Avoid break When
```cpp
// Can use loop condition instead
for (int i = 0; i < 10; i++) {
    if (i == 5) break;  // Better: for (int i = 0; i < 5; i++)
    cout << i << " ";
}

// Complex logic better handled with flag
bool done = false;
while (!done) {
    // Better than while (true) with break
}
```

## Quick Reference

| Context | Effect of break |
|---------|----------------|
| `for` loop | Exits the for loop |
| `while` loop | Exits the while loop |
| `do-while` loop | Exits the do-while loop |
| `switch` | Exits the switch block |
| Nested loops | Exits only innermost loop |

## Summary

- **break** immediately exits the nearest loop or switch
- Cannot be used outside a loop or switch
- In nested loops, breaks only innermost loop
- Use flags or goto to break multiple loops
- Essential for search operations
- Prevents fall-through in switch statements
- Useful in infinite loops with exit conditions
- Makes code more efficient by stopping early
- Place after unreachable code warning
- Alternative: use loop condition when possible

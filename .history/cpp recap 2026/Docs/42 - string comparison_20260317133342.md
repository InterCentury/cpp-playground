# 42 - String Comparison in C++

## What is String Comparison?
String comparison means checking if two strings are equal, or determining their order (which comes first alphabetically). In C++, the `string` class provides multiple ways to compare strings.

## Basic Comparison with Operators

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string s1 = "apple";
    string s2 = "banana";
    string s3 = "apple";
    
    // Equality operators
    if (s1 == s3) {
        cout << "s1 equals s3" << endl;
    }
    
    if (s1 != s2) {
        cout << "s1 is not equal to s2" << endl;
    }
    
    // Relational operators (lexicographic comparison)
    if (s1 < s2) {
        cout << "s1 comes before s2 (alphabetically)" << endl;
    }
    
    if (s2 > s1) {
        cout << "s2 comes after s1" << endl;
    }
    
    return 0;
}
```

## Using compare() Method

The `compare()` method returns:
- **0** if strings are equal
- **Negative value** if first string < second string
- **Positive value** if first string > second string

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string s1 = "apple";
    string s2 = "banana";
    string s3 = "apple";
    string s4 = "Apple";  // Note: uppercase A
    
    // Basic compare
    int result = s1.compare(s2);
    cout << "apple vs banana: " << result << endl;  // negative
    
    result = s1.compare(s3);
    cout << "apple vs apple: " << result << endl;   // 0
    
    result = s2.compare(s1);
    cout << "banana vs apple: " << result << endl;  // positive
    
    // Case-sensitive (ASCII values)
    result = s1.compare(s4);
    cout << "apple vs Apple: " << result << endl;   // positive ('a' > 'A')
    
    return 0;
}
```

## Lexicographic Comparison (Dictionary Order)

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    // Lexicographic (dictionary) order
    string words[] = {"banana", "apple", "cherry", "Apple", "123", "apple"};
    int size = sizeof(words) / sizeof(words[0]);
    
    cout << "Original order:" << endl;
    for (int i = 0; i < size; i++) {
        cout << "  " << words[i] << endl;
    }
    
    // Simple comparison logic
    cout << "\nComparisons (ASCII based):" << endl;
    cout << "apple vs Apple: " << ("apple" < "Apple") << endl;  // false (because 'a' > 'A')
    cout << "apple vs banana: " << ("apple" < "banana") << endl;  // true
    cout << "123 vs apple: " << ("123" < "apple") << endl;  // true (digits < letters)
    
    return 0;
}
```

## Case-Insensitive Comparison

```cpp
#include <iostream>
#include <string>
#include <algorithm>
#include <cctype>
using namespace std;

// Function for case-insensitive comparison
bool caseInsensitiveEqual(const string& s1, const string& s2) {
    if (s1.length() != s2.length()) return false;
    
    for (size_t i = 0; i < s1.length(); i++) {
        if (tolower(s1[i]) != tolower(s2[i])) {
            return false;
        }
    }
    return true;
}

int main() {
    string s1 = "Hello";
    string s2 = "HELLO";
    string s3 = "hello";
    
    // Case-sensitive comparison
    cout << "Case-sensitive:" << endl;
    cout << "Hello == HELLO: " << (s1 == s2) << endl;  // false
    cout << "Hello == hello: " << (s1 == s3) << endl;  // false
    
    // Case-insensitive comparison
    cout << "\nCase-insensitive:" << endl;
    cout << "Hello == HELLO: " << caseInsensitiveEqual(s1, s2) << endl;  // true
    cout << "Hello == hello: " << caseInsensitiveEqual(s1, s3) << endl;  // true
    
    // Using transform (alternative method)
    string lower1 = s1;
    string lower2 = s2;
    transform(lower1.begin(), lower1.end(), lower1.begin(), ::tolower);
    transform(lower2.begin(), lower2.end(), lower2.begin(), ::tolower);
    
    cout << "After transform: " << (lower1 == lower2) << endl;  // true
    
    return 0;
}
```

### Example 2: Sorting Names
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string names[5] = {"John", "Alice", "Bob", "Charlie", "Diana"};
    
    cout << "Original order:" << endl;
    for (int i = 0; i < 5; i++) {
        cout << i+1 << ". " << names[i] << endl;
    }
    
    // Simple bubble sort
    for (int i = 0; i < 4; i++) {
        for (int j = 0; j < 4 - i; j++) {
            if (names[j] > names[j + 1]) {
                // Swap
                string temp = names[j];
                names[j] = names[j + 1];
                names[j + 1] = temp;
            }
        }
    }
    
    cout << "\nSorted order:" << endl;
    for (int i = 0; i < 5; i++) {
        cout << i+1 << ". " << names[i] << endl;
    }
    
    return 0;
}
```

### Example 3: Dictionary Lookup
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string dictionary[] = {"apple", "banana", "cherry", "grape", "orange"};
    string word;
    bool found = false;
    
    cout << "Enter a word to lookup: ";
    cin >> word;
    
    // Convert to lowercase for case-insensitive search
    for (char& c : word) {
        c = tolower(c);
    }
    
    // Linear search
    for (int i = 0; i < 5; i++) {
        if (dictionary[i] == word) {
            found = true;
            cout << "'" << word << "' found at position " << i+1 << endl;
            break;
        }
    }
    
    if (!found) {
        cout << "'" << word << "' not found in dictionary." << endl;
        
        // Find where it would belong
        for (int i = 0; i < 5; i++) {
            if (word < dictionary[i]) {
                cout << "It would be before '" << dictionary[i] << "'" << endl;
                break;
            }
        }
    }
    
    return 0;
}
```

### Example 5: Version Number Comparison
```cpp
#include <iostream>
#include <string>
using namespace std;

// Simple version comparison (e.g., "1.2.3" vs "1.10.1")
int compareVersions(string v1, string v2) {
    int i = 0, j = 0;
    
    while (i < v1.length() || j < v2.length()) {
        // Parse next number from v1
        int num1 = 0;
        while (i < v1.length() && v1[i] != '.') {
            num1 = num1 * 10 + (v1[i] - '0');
            i++;
        }
        i++; // Skip '.'
        
        // Parse next number from v2
        int num2 = 0;
        while (j < v2.length() && v2[j] != '.') {
            num2 = num2 * 10 + (v2[j] - '0');
            j++;
        }
        j++; // Skip '.'
        
        if (num1 < num2) return -1;
        if (num1 > num2) return 1;
    }
    
    return 0;
}

int main() {
    string v1 = "1.2.3";
    string v2 = "1.10.1";
    string v3 = "1.2.4";
    
    int result = compareVersions(v1, v2);
    cout << v1 << " vs " << v2 << ": ";
    if (result < 0) cout << v1 << " < " << v2 << endl;
    else if (result > 0) cout << v1 << " > " << v2 << endl;
    else cout << v1 << " == " << v2 << endl;
    
    result = compareVersions(v1, v3);
    cout << v1 << " vs " << v3 << ": ";
    if (result < 0) cout << v1 << " < " << v3 << endl;
    else if (result > 0) cout << v1 << " > " << v3 << endl;
    else cout << v1 << " == " << v3 << endl;
    
    return 0;
}
```

## Comparing with C-style Strings

```cpp

    // C++ string vs C-style string

        cout << "C++ string equals C-style string" << endl;
    }
    
    if (cppStr == cStr2) {
        cout << "C++ string equals char array" << endl;
```

## Common Mistakes

### Mistake 1: Using == with C-strings (Wrong)
```cpp
char str1[] = "Hello";
char str2[] = "Hello";

// Wrong - compares addresses, not content
if (str1 == str2) {
    cout << "This won't print" << endl;
}

// Correct - use strcmp
if (strcmp(str1, str2) == 0) {
    cout << "Strings are equal" << endl;
}
```

### Mistake 2: Case Sensitivity Surprise
```cpp
string s1 = "Hello";
string s2 = "hello";

// These are different (ASCII: 'H'=72, 'h'=104)
if (s1 == s2) {
    cout << "This won't print" << endl;
}

// If you want case-insensitive, you need to convert
```

### Mistake 3: Ignoring compare() Return Value
```cpp
string s1 = "apple";
string s2 = "banana";

// Wrong assumption
if (s1.compare(s2)) {  // This is true when not equal!
    cout << "This runs when strings are different" << endl;
}

// Correct
if (s1.compare(s2) < 0) {
    cout << "s1 < s2" << endl;
}
if (s1.compare(s2) == 0) {
    cout << "s1 == s2" << endl;
}
```

### Mistake 4: Comparing with Number
```cpp
string s = "123";

// Wrong - compares string with number
if (s == 123) {  // Error!
    cout << "This won't compile" << endl;
}

// Correct - convert first
if (stoi(s) == 123) {
    cout << "Numbers are equal" << endl;
}
```

### Mistake 5: Not Handling Whitespace
```cpp
string s1 = "Hello";
string s2 = "Hello ";  // Note trailing space

// These are different
if (s1 == s2) {
    cout << "This won't print" << endl;
}

// Trim whitespace if needed
while (!s2.empty() && isspace(s2.back())) {
    s2.pop_back();
}
if (s1 == s2) {
    cout << "After trim, they're equal" << endl;
}
```

## Quick Reference

| Comparison | C++ string | C-style string |
|------------|------------|----------------|
| Equality | `s1 == s2` | `strcmp(s1, s2) == 0` |
| Inequality | `s1 != s2` | `strcmp(s1, s2) != 0` |
| Less than | `s1 < s2` | `strcmp(s1, s2) < 0` |
| Greater than | `s1 > s2` | `strcmp(s1, s2) > 0` |
| Case-insensitive | Custom function | `stricmp()` (non-standard) |
| Compare method | `s1.compare(s2)` | `strcmp(s1, s2)` |
| Compare substring | `s1.compare(pos, len, s2)` | `strncmp(s1 + pos, s2, len)` |

## Summary

- **Equality operators** (`==`, `!=`) check if strings are exactly equal
- **Relational operators** (`<`, `>`, `<=`, `>=`) check lexicographic order
- **compare() method** returns negative, zero, or positive for <, ==, >
- Comparisons are **case-sensitive** (based on ASCII values)
- Use `tolower()`/`toupper()` for case-insensitive comparison
- C++ strings work with C-strings in comparisons
- Empty strings can be checked with `empty()` or comparing to `""`
- String literals can be compared directly with string objects
- `strcmp()` is for C-style strings only
- Always consider whitespace in comparisons

# 76 - Member Functions in C++

## What are Member Functions?
Member functions (also called methods) are functions that belong to a class. They define the behavior and operations that can be performed on objects of that class. Member functions have access to the class's private and protected members.

## Type 1: Single File Implementation

```cpp
#include <iostream>
#include <string>
#include <cmath>
using namespace std;

class Circle {
private:
    double radius;
    
public:
    // Member functions defined inside class
    void setRadius(double r) {
        if (r >= 0) radius = r;
    }
    
    double getRadius() const { return radius; }
    double area() const { return 3.14159 * radius * radius; }
    double circumference() const { return 2 * 3.14159 * radius; }
};

int main() {
    Circle c;
    c.setRadius(5.0);
    
    cout << "Radius: " << c.getRadius() << endl;
    cout << "Area: " << c.area() << endl;
    cout << "Circumference: " << c.circumference() << endl;
    
    return 0;
}
```

## Types of Member Functions

### Accessor (Getter) Functions
```cpp
class BankAccount {
private:
    double balance;
    
public:
    // Accessor - returns value without modifying
    double getBalance() const { return balance; }
    
    // Mutator - modifies value
    void deposit(double amount) {
        if (amount > 0) balance += amount;
    }
};
```

### Const Member Functions
```cpp
class Person {
private:
    string name;
    
public:
    // Const member function - promises not to modify object
    string getName() const { return name; }
    
    // Non-const - can modify
    void setName(string n) { name = n; }
};
```

### Static Member Functions
```cpp
class Student {
private:
    static int count;
    
public:
    Student() { count++; }
    
    // Static - belongs to class, not objects
    static int getCount() { return count; }
};

int Student::count = 0;
```

### Inline Member Functions
```cpp
class Math {
public:
    // Implicitly inline (defined inside class)
    int square(int x) { return x * x; }
    
    // Explicitly inline (defined outside)
    inline int cube(int x);
};

inline int Math::cube(int x) { return x * x * x; }
```

## Special Member Functions

### Constructor
```cpp
class Book {
private:
    string title;
    int pages;
    
public:
    // Default constructor
    Book() : title("Unknown"), pages(0) {}
    
    // Parameterized constructor
    Book(string t, int p) : title(t), pages(p) {}
    
    // Copy constructor
    Book(const Book& other) : title(other.title), pages(other.pages) {}
};
```

### Destructor
```cpp
class Resource {
private:
    int* data;
    
public:
    Resource(int size) {
        data = new int[size];
    }
    
    ~Resource() {
        delete[] data;  // Clean up resources
    }
};
```

## Operator Overloading as Member Functions

```cpp
class Complex {
private:
    double real, imag;
    
public:
    Complex(double r = 0, double i = 0) : real(r), imag(i) {}
    
    Complex operator+(const Complex& other) const {
        return Complex(real + other.real, imag + other.imag);
    }
    
    bool operator==(const Complex& other) const {
        return real == other.real && imag == other.imag;
    }
};
```

## Practical Example: Stack Class

### Type 1: Single File
```cpp
#include <iostream>
using namespace std;

class Stack {
private:
    static const int MAX = 100;
    int arr[MAX];
    int top;
    
public:
    Stack() : top(-1) {}
    
    bool isEmpty() const { return top == -1; }
    bool isFull() const { return top == MAX - 1; }
    
    bool push(int value) {
        if (isFull()) return false;
        arr[++top] = value;
        return true;
    }
    
    bool pop(int& value) {
        if (isEmpty()) return false;
        value = arr[top--];
        return true;
    }
    
    void display() const {
        for (int i = top; i >= 0; i--)
            cout << arr[i] << " ";
        cout << endl;
    }
};

int main() {
    Stack s;
    s.push(10);
    s.push(20);
    s.push(30);
    s.display();
    
    int val;
    s.pop(val);
    cout << "Popped: " << val << endl;
    s.display();
    
    return 0;
}
```

## Common Mistakes

### Mistake 1: Forgetting Definition
```cpp
class MyClass {
public:
    void func();  // Declaration only
};
// Missing definition - linker error
```

### Mistake 2: Const Correctness
```cpp
class MyClass {
    int x;
public:
    int getX() { return x; }  // Should be const
};

const MyClass obj;
// obj.getX();  // Error! const object can't call non-const
```

### Mistake 3: Ambiguous Parameter Names
```cpp
void setValue(int value) {
    value = value;  // Sets parameter, not member!
}

// Correct
void setValue(int val) {
    value = val;
}

// Or using this
void setValue(int value) {
    this->value = value;
}
```

## Quick Reference Table

| Member Function Type | Syntax | Key Feature |
|---------------------|--------|-------------|
| Accessor | `int getX() const` | Returns value, read-only |
| Mutator | `void setX(int x)` | Modifies member |
| Const | `void display() const` | Promises no modification |
| Static | `static int count()` | Called without object |
| Inline | `inline int square(int x)` | Suggests code insertion |
| Virtual | `virtual void draw()` | Polymorphism support |
| Constructor | `ClassName()` | Initializes object |
| Destructor | `~ClassName()` | Cleans up resources |

## Summary

- **Member functions** define behavior for class objects
- **Accessors** (getters) return member values; use `const` keyword
- **Mutators** (setters) modify member values; can include validation
- **Const member functions** cannot modify object state
- **Static member functions** belong to class, not individual objects
- **Constructors** initialize objects; **destructors** clean up resources
- **Inline functions** suggest compiler to insert code directly
- **Operator overloading** provides intuitive syntax for objects
- **Virtual functions** enable polymorphism in inheritance hierarchies

## Basic Template

### Type 1: Single File
```cpp
#include <iostream>
using namespace std;

class MyClass {
private:
    int id;
    static int counter;
    
public:
    MyClass() : id(++counter) {}
    
    int getId() const { return id; }
    static int getCounter() { return counter; }
    
    void display() const {
        cout << "ID: " << id << endl;
    }
};

int MyClass::counter = 0;

int main() {
    MyClass obj1, obj2, obj3;
    obj1.display();
    cout << "Total: " << MyClass::getCounter() << endl;
    return 0;
}
```

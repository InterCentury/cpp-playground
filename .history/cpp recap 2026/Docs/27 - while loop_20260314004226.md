# 27 - while Loop in C++

## What is a while Loop?
A `while` loop repeatedly executes a block of code as long as a given condition is true. It's useful when you don't know in advance how many times you need to iterate.

## Basic Syntax

```cpp
while (condition) {
    // code to execute repeatedly
    // update condition variable
}
```

## Simple Example

```cpp
#include <iostream>
using namespace std;

int main() {
    int count = 1;
    
    while (count <= 5) {
        cout << "Count: " << count << endl;
        count++;  // Don't forget to update!
    }
    
    cout << "Loop finished!" << endl;
    
    return 0;
}
```
Output:
```
Count: 1
Count: 2
Count: 3
Count: 4
Count: 5
Loop finished!
```

## How while Loop Works

```cpp
#include <iostream>
using namespace std;

int main() {
    int i = 1;
    
    // Step by step execution
    while (i <= 3) {
        cout << "Iteration " << i << endl;
        i++;
    }
    
    return 0;
}
```
Execution flow:
1. Check i <= 3? (1 <= 3) → true → execute body
2. i becomes 2
3. Check i <= 3? (2 <= 3) → true → execute body
4. i becomes 3
5. Check i <= 3? (3 <= 3) → true → execute body
6. i becomes 4
7. Check i <= 3? (4 <= 3) → false → exit loop

## Practical Examples

```cpp
#include <iostream>
#include <thread>
#include <chrono>
using namespace std;

int main() {
```

```cpp
#include <iostream>
using namespace std;

int main() {
    int n, sum = 0;
```

## Infinite Loops (and How to Avoid Them)

### Common Infinite Loop Mistakes
```cpp
#include <iostream>
using namespace std;

int main() {
    int i = 1;
    
    // Wrong - missing increment
    // while (i <= 5) {
    //     cout << i << " ";  // i never changes → infinite loop
    // }
    
    // Wrong - condition always true
    // while (1) {  // Infinite loop
    //     cout << "Hello ";
    // }
    
    // Correct
    while (i <= 5) {
        cout << i << " ";
        i++;  // Update counter
    }
    cout << endl;
    
    return 0;
}
```

### Controlled Infinite Loop with Break
```cpp
#include <iostream>
using namespace std;

int main() {
    int count = 0;
    
    while (true) {  // Intentional infinite loop
        cout << "Iteration " << count + 1 << endl;
        count++;
        
        if (count >= 5) {
            cout << "Breaking out!" << endl;
            break;  // Exit condition
        }
    }
    
    return 0;
}
```

## while Loop with Strings

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string text = "Hello";
    int i = 0;
    
    cout << "Characters: ";
    while (i < text.length()) {
        cout << text[i] << " ";
        i++;
    }
    cout << endl;
    
    // Count vowels
    i = 0;
    int vowels = 0;
    while (i < text.length()) {
        char ch = tolower(text[i]);
        if (ch == 'a' || ch == 'e' || ch == 'i' || ch == 'o' || ch == 'u') {
            vowels++;
        }
        i++;
    }
    cout << "Vowels: " << vowels << endl;
    
    return 0;
}
```

## while vs do-while

```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 10;
    
    // while - checks condition first
    cout << "while loop:" << endl;
    while (x < 5) {  // Condition false, body never executes
        cout << "This won't print" << endl;
        x++;
    }
    
    // Reset
    x = 10;
    
    // do-while - executes at least once
    cout << "\ndo-while loop:" << endl;
    do {
        cout << "This will print once even though condition is false" << endl;
        x++;
    } while (x < 5);
    
    return 0;
}
```

## Common Mistakes

### Mistake 1: Forgetting to Update Counter
```cpp
int i = 1;

// Wrong - infinite loop
while (i <= 5) {
    cout << i << " ";
    // Missing i++
}

// Correct
while (i <= 5) {
    cout << i << " ";
    i++;  // Update counter
}
```

### Mistake 2: Wrong Condition
```cpp
int i = 1;

// Wrong - loop never runs
while (i > 5) {  // 1 > 5 is false
    cout << i << " ";
    i++;
}

// Correct
while (i <= 5) {
    cout << i << " ";
    i++;
}
```

### Mistake 3: Semicolon After while
```cpp
int i = 1;

// Wrong - semicolon ends the loop
while (i <= 5); {
    cout << i << " ";  // This runs only once, after loop
    i++;
}

// Correct
while (i <= 5) {
    cout << i << " ";
    i++;
}
```

### Mistake 4: Off-by-One Errors
```cpp
int i = 1;

// Wrong - prints only 1-4
while (i < 5) {
    cout << i << " ";  // Prints 1 2 3 4
    i++;
}

// Correct - prints 1-5
while (i <= 5) {
    cout << i << " ";
    i++;
}
```

### Mistake 5: Using Assignment Instead of Comparison
```cpp
int i = 1;

// Wrong - assignment, not comparison
while (i = 5) {  // Always true, infinite loop
    cout << i << " ";
    i++;
}

// Correct
while (i <= 5) {
    cout << i << " ";
    i++;
}
```

## Best Practices

### ✅ Do This
```cpp
// Initialize before loop
int i = 1;

// Clear condition
while (i <= 5) {
    // Do work
    cout << i << " ";
    
    // Update at the end
    i++;
}

// Use meaningful variable names
int studentCount = 0;
while (studentCount < totalStudents) {
    processStudent(studentCount);
    studentCount++;
}

// Handle edge cases
while (num > 0) {
    process(num);
    num--;
}
```

### ❌ Avoid This
```cpp
// Don't forget initialization
int i;
while (i <= 5) { }  // i has garbage value

// Don't use magic numbers
while (x < 10) { }  // What's 10?

// Don't have complex conditions
while (a && b || c && !d || e) { }  // Too complex

// Don't modify loop variable inside multiple places
while (i < 10) {
    i++;  // OK
    if (condition) {
        i += 2;  // Makes loop harder to understand
    }
}
```

## Quick Reference

| Pattern | Description |
|---------|-------------|
| `while (condition) { }` | Basic while loop |
| `while (true) { if (x) break; }` | Infinite loop with break |
| `while (i <= n) { i++; }` | Counting loop |
| `while (cin >> x) { }` | Read until EOF |
| `while (getline(cin, str)) { }` | Read lines until EOF |

## Common Loop Patterns

### Pattern 1: Counter Loop
```cpp
int i = 0;
while (i < n) {
    // Do something
    i++;
}
```

### Pattern 2: Sentinel Loop
```cpp
int value;
cin >> value;
while (value != -1) {
    process(value);
    cin >> value;
}
```

### Pattern 3: Flag Loop
```cpp
bool done = false;
while (!done) {
    // Do something
    if (condition) {
        done = true;
    }
}
```

### Pattern 4: Input Validation Loop
```cpp
int num;
cout << "Enter positive number: ";
cin >> num;

while (num <= 0) {
    cout << "Invalid! Enter again: ";
    cin >> num;
}
```

## Summary

- **while loop** repeats while condition is true
- Condition checked before each iteration
- Must update loop variable inside body
- Can be infinite if condition never becomes false
- Use for unknown number of iterations
- **break** exits loop immediately
- **continue** skips to next iteration
- Watch for off-by-one errors
- Always initialize loop variables
- Indent body for readability

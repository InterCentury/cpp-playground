# 29 - for Loop in C++

## What is for Loop?
The `for` loop is the most commonly used loop in C++. It's perfect when you know exactly how many times you want to iterate. It combines initialization, condition, and increment in one line.

## Basic Syntax

```cpp
for (initialization; condition; update) {
    // code to execute repeatedly
}
```

## Simple Example

```cpp
#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 5; i++) {
        cout << "Iteration: " << i << endl;
    }
    
    return 0;
}
```
Output:
```
Iteration: 1
Iteration: 2
Iteration: 3
Iteration: 4
Iteration: 5
```

## How for Loop Works

```cpp
#include <iostream>
using namespace std;

int main() {
    // Step by step execution
    for (int i = 1; i <= 3; i++) {
        cout << "Iteration " << i << endl;
    }
    
    return 0;
}
```
Execution flow:
1. **Initialization**: `int i = 1` (runs once at start)
2. **Condition check**: `i <= 3`? (1 <= 3) → true → execute body
3. **Body executes**: prints "Iteration 1"
4. **Update**: `i++` → i becomes 2
5. **Condition check**: `i <= 3`? (2 <= 3) → true → execute body
6. **Body executes**: prints "Iteration 2"
7. **Update**: `i++` → i becomes 3
8. **Condition check**: `i <= 3`? (3 <= 3) → true → execute body
9. **Body executes**: prints "Iteration 3"
10. **Update**: `i++` → i becomes 4
11. **Condition check**: `i <= 3`? (4 <= 3) → false → exit loop

## Parts of for Loop

```cpp
for (int i = 1; i <= 5; i++) {
//    ^          ^         ^
//    |          |         |
// initialization condition update
//    |          |         |
//  runs once  checked   runs after
//  at start   before    each body
//             each body  execution
```

## Different Variations

### Counting Up
```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Counting up: ";
    for (int i = 1; i <= 5; i++) {
        cout << i << " ";
    }
    cout << endl;
    
    return 0;
}
```

### Counting Down
```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Counting down: ";
    for (int i = 5; i >= 1; i--) {
        cout << i << " ";
    }
    cout << endl;
    
    return 0;
}
```

### Step by 2
```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Even numbers: ";
    for (int i = 2; i <= 10; i += 2) {
        cout << i << " ";
    }
    cout << endl;
    
    return 0;
}
```

### Multiple Variables
```cpp
#include <iostream>
using namespace std;

int main() {
    for (int i = 1, j = 10; i <= 5; i++, j--) {
        cout << "i = " << i << ", j = " << j << endl;
    }
    
    return 0;
}
```

## Practical Examples

### Example 1: Sum of Numbers
```cpp
#include <iostream>
using namespace std;

int main() {
    int n, sum = 0;
    
    cout << "Enter a number: ";
    cin >> n;
    
    for (int i = 1; i <= n; i++) {
        sum += i;
    }
    
    cout << "Sum of 1 to " << n << " = " << sum << endl;
    
    return 0;
}
```

### Example 2: Factorial
```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    long long factorial = 1;
    
    cout << "Enter a number: ";
    cin >> n;
    
    for (int i = 1; i <= n; i++) {
        factorial *= i;
    }
    
    cout << n << "! = " << factorial << endl;
    
    return 0;
}
```

### Example 3: Multiplication Table
```cpp
#include <iostream>
using namespace std;

int main() {
    int num;
    
    cout << "Enter a number: ";
    cin >> num;
    
    cout << "Multiplication table of " << num << ":" << endl;
    for (int i = 1; i <= 10; i++) {
        cout << num << " × " << i << " = " << num * i << endl;
    }
    
    return 0;
}
```

### Example 4: Print Array Elements
```cpp
#include <iostream>
using namespace std;

int main() {
    int numbers[] = {10, 20, 30, 40, 50};
    int size = sizeof(numbers) / sizeof(numbers[0]);
    
    cout << "Array elements: ";
    for (int i = 0; i < size; i++) {
        cout << numbers[i] << " ";
    }
    cout << endl;
    
    // Sum of array
    int sum = 0;
    for (int i = 0; i < size; i++) {
        sum += numbers[i];
    }
    cout << "Sum: " << sum << endl;
    
    return 0;
}
```

### Example 5: Find Maximum in Array
```cpp
#include <iostream>
using namespace std;

int main() {
    int numbers[] = {45, 23, 78, 12, 89, 34};
    int size = sizeof(numbers) / sizeof(numbers[0]);
    int max = numbers[0];
    
    for (int i = 1; i < size; i++) {
        if (numbers[i] > max) {
            max = numbers[i];
        }
    }
    
    cout << "Maximum value: " << max << endl;
    
    return 0;
}
```

## Nested for Loops

### Example 1: Rectangle Pattern
```cpp
#include <iostream>
using namespace std;

int main() {
    int rows = 3, cols = 5;
    
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < cols; j++) {
            cout << "* ";
        }
        cout << endl;
    }
    
    return 0;
}
```
Output:
```
* * * * *
* * * * *
* * * * *
```

### Example 2: Triangle Pattern
```cpp
#include <iostream>
using namespace std;

int main() {
    int height = 5;
    
    for (int i = 1; i <= height; i++) {
        for (int j = 1; j <= i; j++) {
            cout << "* ";
        }
        cout << endl;
    }
    
    return 0;
}
```
Output:
```
* 
* * 
* * * 
* * * * 
* * * * *
```

### Example 3: Multiplication Table Grid

#include <iostream>

## for Loop with Strings

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string text = "Hello";
    
    cout << "Characters: ";
    for (int i = 0; i < text.length(); i++) {
        cout << text[i] << " ";
    }
    cout << endl;
    
    // Count vowels
    int vowels = 0;
    for (int i = 0; i < text.length(); i++) {
        char ch = tolower(text[i]);
        if (ch == 'a' || ch == 'e' || ch == 'i' || ch == 'o' || ch == 'u') {
            vowels++;
        }
    }
    cout << "Vowels: " << vowels << endl;
    
    return 0;
}
```

## Range-based for Loop (C++11)

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int numbers[] = {10, 20, 30, 40, 50};
    
    cout << "Traditional for loop: ";
    for (int i = 0; i < 5; i++) {
        cout << numbers[i] << " ";
    }
    cout << endl;
    
    cout << "Range-based for loop: ";
    for (int num : numbers) {
        cout << num << " ";
    }
    cout << endl;
    
    // With vector
    vector<string> fruits = {"apple", "banana", "orange"};
    cout << "Fruits: ";
    for (const string& fruit : fruits) {
        cout << fruit << " ";
    }
    cout << endl;
    
    return 0;
}
```

## Infinite for Loop

```cpp
#include <iostream>
using namespace std;

int main() {
    // Infinite loop (usually with break)
    int count = 0;
    
    for (;;) {
        cout << "Iteration " << ++count << endl;
        
        if (count >= 5) {
            break;  // Exit condition
        }
    }
    
    return 0;
}
```

## for Loop with break and continue

### Using break
```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Break at 3:" << endl;
    for (int i = 1; i <= 5; i++) {
        if (i == 3) {
            cout << "Breaking!" << endl;
            break;
        }
        cout << "i = " << i << endl;
    }
    
    return 0;
}
```

### Using continue
```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Skip 3:" << endl;
    for (int i = 1; i <= 5; i++) {
        if (i == 3) {
            cout << "Skipping!" << endl;
            continue;
        }
        cout << "i = " << i << endl;
    }
    
    return 0;
}
```

## Common Mistakes

### Mistake 1: Off-by-One Errors
```cpp
// Wrong - prints 1-4 only
for (int i = 1; i < 5; i++) {
    cout << i << " ";  // Prints 1 2 3 4
}

// Correct - prints 1-5
for (int i = 1; i <= 5; i++) {
    cout << i << " ";
}
```

### Mistake 2: Semicolon After for
```cpp
// Wrong - semicolon ends the loop
for (int i = 1; i <= 5; i++); {
    cout << i << " ";  // Error! i not in scope
}

// Correct
for (int i = 1; i <= 5; i++) {
    cout << i << " ";
}
```

### Mistake 3: Using = Instead of ==
```cpp
// Wrong - infinite loop
for (int i = 1; i = 5; i++) {  // Assignment, not comparison
    cout << i << " ";
}

// Correct
for (int i = 1; i <= 5; i++) {
    cout << i << " ";
}
```

### Mistake 4: Modifying Loop Variable Inside
```cpp
// Wrong - confusing
for (int i = 1; i <= 5; i++) {
    cout << i << " ";
    i++;  // Don't do this! i increments twice
}

// Correct
for (int i = 1; i <= 5; i++) {
    cout << i << " ";
}
```

### Mistake 5: Wrong Variable Scope
```cpp
// Wrong - i not accessible outside
for (int i = 1; i <= 5; i++) {
    cout << i << " ";
}
cout << "Final i: " << i;  // Error! i not declared

// Correct
int i;
for (i = 1; i <= 5; i++) {
    cout << i << " ";
}
cout << "Final i: " << i;  // OK
```

## for Loop Variations

### Empty Initialization
```cpp
int i = 1;
for (; i <= 5; i++) {
    cout << i << " ";
}
```

### Empty Condition (Infinite)
```cpp
for (int i = 1; ; i++) {
    cout << i << " ";
    if (i >= 5) break;
}
```

### Empty Update
```cpp
for (int i = 1; i <= 5; ) {
    cout << i << " ";
    i++;  // Update inside
}
```

### Multiple Updates
```cpp
for (int i = 1, j = 10; i <= 5; i++, j--) {
    cout << "i=" << i << ", j=" << j << endl;
}
```

## Quick Reference

| Pattern | Description |
|---------|-------------|
| `for (int i = 0; i < n; i++)` | Count from 0 to n-1 |
| `for (int i = 1; i <= n; i++)` | Count from 1 to n |
| `for (int i = n; i >= 1; i--)` | Count down from n to 1 |
| `for (int i = 0; i < n; i += 2)` | Step by 2 |
| `for (int x : array)` | Range-based loop |
| `for (;;)` | Infinite loop |

## Comparison with Other Loops

| Feature | for | while | do-while |
|---------|-----|-------|----------|
| Known iterations | ✅ Best | ⚠️ OK | ⚠️ OK |
| Unknown iterations | ⚠️ OK | ✅ Best | ✅ Best |
| Must run at least once | ❌ No | ❌ No | ✅ Yes |
| Compact syntax | ✅ Yes | ⚠️ OK | ⚠️ OK |
| Array traversal | ✅ Yes | ✅ Yes | ⚠️ OK |

## Summary

- **for loop** combines init, condition, update in one line
- Perfect for counting and known iterations
- **Initialization** runs once at start
- **Condition** checked before each iteration
- **Update** runs after each iteration
- Use for array/string traversal
- Nested for loops create patterns
- Range-based for (C++11+) simplifies array iteration
- Watch for off-by-one errors
- Can be infinite with `for(;;)`

## Basic Template
```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    
    cout << "Enter number of iterations: ";
    cin >> n;
    
    // Counting up
    cout << "Counting up: ";
    for (int i = 1; i <= n; i++) {
        cout << i << " ";
    }
    cout << endl;
    
    // Counting down
    cout << "Counting down: ";
    for (int i = n; i >= 1; i--) {
        cout << i << " ";
    }
    cout << endl;
    
    // Sum
    int sum = 0;
    for (int i = 1; i <= n; i++) {
        sum += i;
    }
    cout << "Sum 1 to " << n << ": " << sum << endl;
    
    return 0;
}
```

# 40 - C++ string Class

## What is the C++ string Class?
The `string` class is a modern C++ way to handle text. It's part of the Standard Library and provides a safe, convenient alternative to C-style strings.

## Basic Syntax

```cpp
#include <string>
using namespace std;

string variableName;           // Empty string
string variableName = "text";  // Initialize with text
string variableName("text");   // Constructor initialization
```

## Simple Example

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string name = "John";
    string greeting = "Hello";
    
    cout << greeting << ", " << name << "!" << endl;
    cout << "Name length: " << name.length() << endl;
    
    return 0;
}
```

## Different Ways to Create Strings

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    // Empty string
    string s1;
    
    // Initialize with string literal
    string s2 = "Hello";
    string s3("World");
    
    // Initialize with character
    string s4(5, 'A');  // "AAAAA"
    
    // Copy from another string
    string s5 = s2;     // "Hello"
    
    // Substring
    string s6 = s2.substr(1, 3);  // "ell" (from index 1, length 3)
    
    cout << "s1: [" << s1 << "]" << endl;
    cout << "s2: " << s2 << endl;
    cout << "s3: " << s3 << endl;
    cout << "s4: " << s4 << endl;
    cout << "s5: " << s5 << endl;
    cout << "s6: " << s6 << endl;
    
    return 0;
}
```

### Length and Size
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string text = "Hello World";
    
    cout << "String: " << text << endl;
    cout << "Length: " << text.length() << endl;  // 11

    return 0;
}
```

### Accessing Characters
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string text = "Hello";
    
    // Using array index []
    cout << "First character: " << text[0] << endl;     // H
    cout << "Second character: " << text[1] << endl;   // e
    
    // Using at() (with bounds checking)
    cout << "Last character: " << text.at(4) << endl;  // o
    
    // Modify characters
    text[0] = 'J';
    cout << "Modified: " << text << endl;  // Jello
    
    // Loop through characters
    cout << "Characters: ";
    for (char c : text) {
        cout << c << " ";
    }
    cout << endl;
    
    return 0;
}
```

## Practical Examples

### Example 1: Reverse String
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string text;
    
    cout << "Enter a string: ";
    getline(cin, text);
    
    cout << "Original: " << text << endl;
    
    // Reverse using loop
    cout << "Reversed: ";
    for (int i = text.length() - 1; i >= 0; i--) {
        cout << text[i];
    }
    cout << endl;
    
    // Alternative using reverse iterator
    string reversed(text.rbegin(), text.rend());
    cout << "Reversed (iterator): " << reversed << endl;
    
    return 0;
}
```

### Example 2: Palindrome Checker
```cpp
#include <iostream>
#include <string>
#include <algorithm>
using namespace std;

int main() {
    string text;
    
    cout << "Enter a word: ";
    cin >> text;
    
    // Convert to lowercase for case-insensitive check
    string lower = text;
    transform(lower.begin(), lower.end(), lower.begin(), ::tolower);
    
    // Check palindrome
    bool isPalindrome = true;
    int len = lower.length();
    
    for (int i = 0; i < len / 2; i++) {
        if (lower[i] != lower[len - 1 - i]) {
            isPalindrome = false;
            break;
        }
    }
    
    cout << text << " is ";
    if (!isPalindrome) cout << "not ";
    cout << "a palindrome" << endl;
    
    return 0;
}
```

### Example 3: Word Counter
```cpp
#include <iostream>
#include <string>
#include <sstream>
using namespace std;

int main() {
    string sentence;
    
    cout << "Enter a sentence: ";
    getline(cin, sentence);
    
    // Count words using stringstream
    stringstream ss(sentence);
    string word;
    int count = 0;
    
    while (ss >> word) {
        count++;
    }
    
    cout << "Word count: " << count << endl;
    
    // Character count (excluding spaces)
    int charCount = 0;
    for (char c : sentence) {
        if (c != ' ') charCount++;
    }
    cout << "Character count (no spaces): " << charCount << endl;
    
    return 0;
}
```

### Example 4: Email Validator (Simple)
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string email;
    
    cout << "Enter email address: ";
    getline(cin, email);
    
    bool hasAt = false;
    bool hasDot = false;
    int atPos = -1;
    
    // Check for @
    for (int i = 0; i < email.length(); i++) {
        if (email[i] == '@') {
            hasAt = true;
            atPos = i;
            break;
        }
    }
    
    // Check for . after @
    if (hasAt && atPos != -1) {
        for (int i = atPos + 1; i < email.length(); i++) {
            if (email[i] == '.') {
                hasDot = true;
                break;
            }
        }
    }
    
    if (hasAt && hasDot && atPos > 0) {
        cout << "Valid email format" << endl;
    } else {
        cout << "Invalid email format" << endl;
    }
    
    return 0;
}
```

### Example 5: Name Formatter
```cpp
#include <iostream>
#include <string>
#include <algorithm>
using namespace std;

int main() {
    string firstName, lastName;
    
    cout << "Enter first name: ";
    cin >> firstName;
    cout << "Enter last name: ";
    cin >> lastName;
    
    // Capitalize first letters
    if (!firstName.empty()) {
        firstName[0] = toupper(firstName[0]);
    }
    if (!lastName.empty()) {
        lastName[0] = toupper(lastName[0]);
    }
    
    // Make rest lowercase
    for (int i = 1; i < firstName.length(); i++) {
        firstName[i] = tolower(firstName[i]);
    }
    for (int i = 1; i < lastName.length(); i++) {
        lastName[i] = tolower(lastName[i]);
    }
    
    // Display formats
    cout << "\nFormatted name:" << endl;
    cout << "Full: " << firstName << " " << lastName << endl;
    cout << "Last, First: " << lastName << ", " << firstName << endl;
    cout << "Initials: " << firstName[0] << lastName[0] << endl;
    cout << "Username: " << tolower(firstName[0]) << lastName << endl;
    
    return 0;
}
```

## String Methods

### Searching
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string text = "Hello world, welcome to C++";
    
    // find - returns position (or string::npos if not found)
    size_t pos = text.find("world");
    if (pos != string::npos) {
        cout << "'world' found at position: " << pos << endl;
    }
    
    // find from specific position
    pos = text.find("o", 5);  // Find 'o' starting from index 5
    cout << "Next 'o' at: " << pos << endl;
    
    // rfind - reverse find (from end)
    pos = text.rfind("o");
    cout << "Last 'o' at: " << pos << endl;
    
    // find_first_of - any of the characters
    pos = text.find_first_of("aeiou");  // First vowel
    cout << "First vowel at: " << pos << endl;
    
    return 0;
}
```

### Modifying
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string text = "Hello World";
    
    // Insert
    text.insert(5, " beautiful");
    cout << "After insert: " << text << endl;  // Hello beautiful World
    
    // Erase
    text.erase(5, 10);  // Remove 10 characters from position 5
    cout << "After erase: " << text << endl;   // Hello World
    
    // Replace
    text.replace(6, 5, "C++");
    cout << "After replace: " << text << endl; // Hello C++
    
    // Append
    text.append(" Programmers");
    cout << "After append: " << text << endl;  // Hello C++ Programmers
    
    // Clear
    text.clear();
    cout << "After clear: [" << text << "]" << endl;
    
    return 0;
}
```

## String Comparison

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string s1 = "apple";
    string s2 = "banana";
    string s3 = "apple";
    
    // Direct comparison with operators
    if (s1 == s3) {
        cout << "s1 equals s3" << endl;
    }
    
    if (s1 != s2) {
        cout << "s1 is not equal to s2" << endl;
    }
    
    if (s1 < s2) {
        cout << "s1 comes before s2 (lexicographically)" << endl;
    }
    
    // Using compare() method
    int result = s1.compare(s2);
    if (result < 0) {
        cout << "s1 < s2" << endl;
    } else if (result > 0) {
        cout << "s1 > s2" << endl;
    } else {
        cout << "s1 == s2" << endl;
    }
    
    return 0;
}
```

## Common String Methods Reference

| Method | Description | Example |
|--------|-------------|---------|
| `length()` / `size()` | Returns string length | `s.length()` |
| `empty()` | Checks if string is empty | `s.empty()` |
| `clear()` | Empties the string | `s.clear()` |
| `at(pos)` | Character at position (checked) | `s.at(5)` |
| `front()` | First character | `s.front()` |
| `back()` | Last character | `s.back()` |
| `append(str)` | Appends string | `s.append("!")` |
| `insert(pos, str)` | Inserts at position | `s.insert(5, "hi")` |
| `erase(pos, len)` | Erases characters | `s.erase(5, 3)` |
| `replace(pos, len, str)` | Replaces characters | `s.replace(0, 2, "new")` |
| `find(str)` | Finds substring | `s.find("lo")` |
| `rfind(str)` | Finds from end | `s.rfind("o")` |
| `substr(pos, len)` | Extracts substring | `s.substr(1, 3)` |
| `c_str()` | Converts to C-string | `s.c_str()` |
| `stoi()` / `stod()` | String to number | `int n = stoi("123");` |
| `to_string()` | Number to string | `string s = to_string(42);` |

## Common Mistakes

### Mistake 1: Not Including <string> Header
```cpp
// Wrong
string name = "John";  // Error: string not defined

// Correct
#include <string>
using namespace std;
string name = "John";
```

### Mistake 2: Mixing cin and getline Without ignore
```cpp
int age;
string name;

cout << "Enter age: ";
cin >> age;  // Leaves newline in buffer
cout << "Enter name: ";
getline(cin, name);  // Reads the leftover newline

// Correct
cin >> age;
cin.ignore();  // Clear the buffer
getline(cin, name);
```

### Mistake 3: Using C-string Functions on C++ Strings
```cpp
string s = "Hello";

// Wrong - strlen expects char*
int len = strlen(s);  // Error!

// Correct
int len = s.length();
```

### Mistake 4: Forgetting string::npos
```cpp
string s = "Hello";
size_t pos = s.find("x");

// Wrong - npos is large, not -1
if (pos == -1) { }  // Works but not portable

// Correct
if (pos == string::npos) {
    cout << "Not found" << endl;
}
```

### Mistake 5: Modifying During Iteration
```cpp
string s = "Hello";

// Wrong - modifying while iterating
for (char c : s) {
    if (c == 'l') {
        s.replace(0, 1, "L");  // Dangerous!
    }
}

// Better - use index-based loop
for (int i = 0; i < s.length(); i++) {
    if (s[i] == 'l') {
        s[i] = 'L';
    }
}
```

## C++ string vs C-style String

| Feature | C++ string | C-style string |
|---------|------------|----------------|
| Header | `<string>` | `<cstring>` |
| Memory management | Automatic | Manual |
| Bounds checking | Yes (with at()) | No |
| Assignment | `=` operator | `strcpy()` |
| Comparison | `==`, `<`, etc. | `strcmp()` |
| Concatenation | `+` operator | `strcat()` |
| Length | `.length()` | `strlen()` |
| Modify | Easy | Careful needed |
| Performance | Good | Very good |
| Safety | High | Low |

## Quick Reference

| Operation | C++ string | C-style string |
|-----------|------------|----------------|
| Declare | `string s;` | `char s[100];` |
| Initialize | `string s = "Hi";` | `char s[] = "Hi";` |
| Input | `getline(cin, s);` | `cin.getline(s, 100);` |
| Output | `cout << s;` | `cout << s;` |
| Length | `s.length()` | `strlen(s)` |
| Copy | `s2 = s1;` | `strcpy(s2, s1);` |
| Compare | `if (s1 == s2)` | `if (strcmp(s1, s2) == 0)` |
| Concatenate | `s3 = s1 + s2;` | `strcat(s1, s2);` |
| Access char | `s[0]` or `s.at(0)` | `s[0]` |
| Find | `s.find("lo")` | `strstr(s, "lo")` |

## Summary

- **string class** is modern C++ way to handle text
- Automatic memory management
- Safe with bounds checking (using `at()`)
- Supports intuitive operators (`+`, `==`, `=`)
- Rich set of methods for manipulation
- Use `getline()` for input with spaces
- Convert with C-strings using `c_str()`
- Convert to numbers with `stoi()`, `stod()`
- Always include `<string>` header
- More convenient and safer than C-style strings

## Basic Template
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string name;
    string greeting;
    
    // Input
    cout << "Enter your name: ";
    getline(cin, name);
    
    // String operations
    greeting = "Hello, " + name + "!";
    
    // Output
    cout << greeting << endl;
    cout << "Your name has " << name.length() << " characters" << endl;
    
    // Character access
    if (!name.empty()) {
        cout << "First character: " << name[0] << endl;
        cout << "Last character: " << name[name.length() - 1] << endl;
    }
    
    // Find
    size_t pos = name.find(' ');
    if (pos != string::npos) {
        cout << "Space found at position: " << pos << endl;
    }
    
    return 0;
}
```

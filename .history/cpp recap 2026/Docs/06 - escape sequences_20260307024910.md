# 06 - Escape Sequences in C++

## What are Escape Sequences?
Escape sequences are special characters used in strings that perform specific actions. They start with a backslash `\` followed by a character.

## Why Use Escape Sequences?
- To insert characters that are difficult to type directly
- To format text output
- To represent special characters

## List of Common Escape Sequences

| Sequence | Name | What it does |
|----------|------|--------------|
| `\n` | Newline | Moves cursor to next line |
| `\t` | Tab | Adds horizontal tab space |
| `\\` | Backslash | Prints a single backslash |
| `\"` | Double quote | Prints double quote |
| `\'` | Single quote | Prints single quote |
| `\0` | Null | String terminator |
| `\r` | Carriage return | Moves cursor to line start |
| `\b` | Backspace | Moves cursor back one space |
| `\a` | Alert/Beep | Makes computer beep sound |

## Detailed Examples

### 1. Newline `\n`
```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "First line\nSecond line\nThird line";
    return 0;
}
```
Output:
```
First line
Second line
Third line
```

### 2. Tab `\t`
```cpp
cout << "Name\tAge\tCity\n";
cout << "John\t25\tNYC\n";
cout << "Anna\t30\tLA\n";
```
Output:
```
Name    Age    City
John    25     NYC
Anna    30     LA
```

### 3. Backslash `\\`
```cpp
cout << "C:\\Users\\John\\Documents";
```
Output:
```
C:\Users\John\Documents
```

### 4. Double Quote `\"`
```cpp
cout << "She said, \"Hello!\"";
```
Output:
```
She said, "Hello!"
```

### 5. Single Quote `\'`
```cpp
cout << "It\'s a nice day";
```
Output:
```
It's a nice day
```

### 6. Alert/Beep `\a`
```cpp
cout << "Warning!\a\a\a";
// Makes 3 beep sounds
```

### 7. Backspace `\b`
```cpp
cout << "Hello\bWorld";
```
Output:
```
HellWorld  (removes 'o')
```

### 8. Carriage Return `\r`
```cpp
cout << "12345\rABCDE";
```
Output:
```
ABCDE  (overwrites beginning)
```

## Practical Examples

### Example 1: Formatting a Table
```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Product\tPrice\tQuantity\tTotal\n";
    cout << "Laptop\t$999.99\t2\t\t$1999.98\n";
    cout << "Mouse\t$24.50\t5\t\t$122.50\n";
    cout << "Keyboard$79.99\t1\t\t$79.99\n";
    
    return 0;
}
```
Output:
```
Product Price   Quantity    Total
Laptop  $999.99 2           $1999.98
Mouse   $24.50  5           $122.50
Keyboard$79.99  1           $79.99
```

### Example 2: File Path Display

### Use Case 1: File Paths

```cpp

```


## Escape Sequence vs endl

| Feature | `\n` | `endl` |
|---------|------|--------|
| What it does | New line | New line + flush buffer |
| Speed | Faster | Slower |
| Usage | Most cases | When need immediate output |
| Example | `cout << "Hi\n";` | `cout << "Hi" << endl;` |

## Common Mistakes

### Mistake 1: Wrong slash
```cpp
// Wrong
cout << "Hello /n World";

// Correct
cout << "Hello \n World";
```

### Mistake 2: Forgetting backslash
```cpp
// Wrong
cout << "Path: C:Users";

// Correct
cout << "Path: C:\\Users";
```

### Mistake 3: Spaces inside sequence
```cpp
// Wrong
cout << "Hello \ n World";

// Correct
cout << "Hello \n World";
```

### Mistake 4: Using single quotes for string
```cpp
// Wrong
cout << '\nHello\n';

// Correct
cout << "\nHello\n";
```

### Mistake 5: Multiple backslashes confusion
```cpp
// Wrong - prints one \
cout << "C:\Users";

// Correct - prints C:\Users
cout << "C:\\Users";

// Also correct - raw string literal
cout << R"(C:\Users)";
```

## Raw String Literals (C++11)

```cpp
// Instead of escape sequences
cout << "C:\\Users\\Name\\Documents";

// Use raw string literal
cout << R"(C:\Users\Name\Documents)";

// With quotes inside
cout << R"(He said "Hello")";
```

## Summary

- Escape sequences start with backslash `\`
- `\n` creates new line (most commonly used)
- `\t` adds tab space for alignment
- `\\` prints actual backslash
- `\"` and `\'` print quotes inside strings
- Use `\a` for alert/beep sound
- `\b` moves cursor back (deletes previous character)
- `\r` moves cursor to line start
- Multiple escape sequences can be combined
- Raw strings `R"(text)"` avoid escape sequences
- Different from endl (endl also flushes buffer)
- Essential for formatting output properly

## Basic Template
```cpp
#include <iostream>
using namespace std;

int main() {
    // New lines
    cout << "Line1\nLine2\nLine3\n";
    
    // Tabs for formatting
    cout << "Col1\tCol2\tCol3\n";
    
    // Special characters
    cout << "Path: C:\\Program Files\n";
    cout << "Quote: \"Hello\"\n";
    cout << "Single: \'A\'\n";
    
    // Combined
    cout << "Name:\tJohn\t\tAge:\t25\n";
    
    return 0;
}
```

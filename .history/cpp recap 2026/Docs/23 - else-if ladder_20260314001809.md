# 23 - else-if Ladder in C++

## What is else-if Ladder?
The `else-if` ladder allows you to check multiple conditions in sequence. It's useful when you have several mutually exclusive conditions to test.

## Basic Syntax

```cpp
if (condition1) {
    // code executes if condition1 is true
} else if (condition2) {
    // code executes if condition1 is false and condition2 is true
} else if (condition3) {
    // code executes if condition1 and condition2 are false and condition3 is true
} else {
    // code executes if all conditions are false
}
```

## Simple Example

```cpp
#include <iostream>
using namespace std;

int main() {
    int score = 85;
    
    if (score >= 90) {
        cout << "Grade: A" << endl;
    } else if (score >= 80) {
        cout << "Grade: B" << endl;
    } else if (score >= 70) {
        cout << "Grade: C" << endl;
    } else if (score >= 60) {
        cout << "Grade: D" << endl;
    } else {
        cout << "Grade: F" << endl;
    }
    
    return 0;
}
```
Output:
```
Grade: B
```

## Multiple Conditions in else-if

## Important Characteristics

```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 5;
    
    // Only first matching condition executes
    if (x > 0) {
        cout << "Positive" << endl;  // This runs
    } else if (x > 3) {
        cout << "Greater than 3" << endl;  // This won't run
    } else if (x > 1) {
        cout << "Greater than 1" << endl;  // This won't run
    }
    
    // Order matters!
    if (x > 3) {
        cout << "Greater than 3" << endl;  // This runs
    } else if (x > 0) {
        cout << "Positive" << endl;  // This won't run
    }
    
    return 0;
}
```

## Common Mistakes

### Mistake 1: Wrong Order of Conditions
```cpp
int score = 85;

// Wrong - first condition catches all
if (score >= 60) {
    cout << "Passing" << endl;  // This runs for 85
} else if (score >= 80) {
    cout << "Good grade" << endl;  // Never runs for any score >=60
}

// Correct
if (score >= 80) {
    cout << "Good grade" << endl;
} else if (score >= 60) {
    cout << "Passing" << endl;
}
```

### Mistake 2: Using else-if Without if
```cpp
// Wrong
else if (x > 5) {
    cout << "x > 5" << endl;
}

// Correct
if (x > 5) {
    cout << "x > 5" << endl;
} else if (x > 3) {
    cout << "x > 3" << endl;
}
```

### Mistake 3: Missing Braces
```cpp
int x = 10;

// Risky - only first line is in else-if
if (x > 10)
    cout << "Greater" << endl;
else if (x < 10)
    cout << "Less" << endl;
    cout << "This always runs!" << endl;  // Not in else-if!

// Correct
if (x > 10) {
    cout << "Greater" << endl;
} else if (x < 10) {
    cout << "Less" << endl;
    cout << "This runs only when x < 10" << endl;
}
```

### Mistake 4: Semicolon After Condition
```cpp
int x = 5;

// Wrong
if (x > 3); {
    cout << "x > 3" << endl;  // Always runs!
} else if (x > 1) {
    cout << "x > 1" << endl;  // Error!
}

// Correct
if (x > 3) {
    cout << "x > 3" << endl;
} else if (x > 1) {
    cout << "x > 1" << endl;
}
```

### Mistake 5: Overlapping Conditions
```cpp
int x = 5;

// Inefficient - conditions overlap
if (x > 0) {
    cout << "Positive" << endl;
} else if (x > 3) {  // This condition is never checked for positive numbers
    cout << "Greater than 3" << endl;
}

// Better
if (x > 3) {
    cout << "Greater than 3" << endl;
} else if (x > 0) {
    cout << "Positive but ≤ 3" << endl;
}
```

## vs Multiple if Statements

```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 5;
    
    cout << "Using else-if ladder:" << endl;
    if (x > 0) {
        cout << "Positive" << endl;
    } else if (x < 0) {
        cout << "Negative" << endl;
    } else {
        cout << "Zero" << endl;
    }
    // Only one message prints
    
    cout << "\nUsing multiple ifs:" << endl;
    if (x > 0) {
        cout << "Positive" << endl;
    }
    if (x < 0) {
        cout << "Negative" << endl;
    }
    if (x == 0) {
        cout << "Zero" << endl;
    }
    // Still only one message, but all conditions checked
    
    return 0;
}
```

## Quick Reference

| Code | Meaning |
|------|---------|
| `if (c1) { } else if (c2) { }` | Check c1, then c2 if c1 false |
| `if (c1) { } else if (c2) { } else { }` | Default when all false |
| Order matters | Most specific first |
| Only one block executes | First true condition wins |

## Summary

- **else-if ladder** checks multiple conditions in sequence
- Only the first true condition executes
- Order matters - put most specific conditions first
- Can have multiple else-if blocks
- Optional else at the end catches all false cases
- More efficient than multiple if statements
- Conditions are mutually exclusive in execution
- Use for multi-way decisions
- Always indent properly for readability

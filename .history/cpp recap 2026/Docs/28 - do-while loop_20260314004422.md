# 28 - do-while Loop in C++

## What is do-while Loop?
A `do-while` loop is similar to a while loop, but it guarantees that the loop body executes at least once because the condition is checked after the body.

## Basic Syntax

```cpp
do {
    // code to execute
    // update condition variable
} while (condition);
```

## Simple Example

```cpp
#include <iostream>
using namespace std;

int main() {
    int count = 1;
    
    do {
        cout << "Count: " << count << endl;
        count++;
    } while (count <= 5);
    
    cout << "Loop finished!" << endl;
    
    return 0;
}
```
Output:
```
Count: 1
Count: 2
Count: 3
Count: 4
Count: 5
Loop finished!
```

## while vs do-while (Key Difference)

```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 10;
    
    // while - checks condition first
    cout << "while loop:" << endl;
    while (x < 5) {  // Condition false, body never executes
        cout << "This won't print" << endl;
        x++;
    }
    cout << "while finished" << endl;
    
    // Reset
    x = 10;
    
    // do-while - executes at least once
    cout << "\ndo-while loop:" << endl;
    do {
        cout << "This prints once even though condition is false" << endl;
        x++;
    } while (x < 5);
    cout << "do-while finished" << endl;
    
    return 0;
}
```

## How do-while Works

```cpp
#include <iostream>
using namespace std;

int main() {
    int i = 1;
    
    // Step by step execution
    do {
        cout << "Iteration " << i << endl;
        i++;
    } while (i <= 3);
    
    return 0;
}
```
Execution flow:
1. Execute body (i=1) → print, i becomes 2
2. Check condition (2 <= 3) → true → execute body again
3. Execute body (i=2) → print, i becomes 3
4. Check condition (3 <= 3) → true → execute body again
5. Execute body (i=3) → print, i becomes 4
6. Check condition (4 <= 3) → false → exit loop

## Practical Examples

### Example 1: Menu System (Perfect for do-while)
```cpp
#include <iostream>
using namespace std;

int main() {
    int choice;
    
    do {
        cout << "\n=== MAIN MENU ===" << endl;
        cout << "1. Start Game" << endl;
        cout << "2. Load Game" << endl;
        cout << "3. Settings" << endl;
        cout << "4. Exit" << endl;
        cout << "Choice: ";
        cin >> choice;
        
        switch (choice) {
            case 1:
                cout << "Game started!" << endl;
                break;
            case 2:
                cout << "Game loaded!" << endl;
                break;
            case 3:
                cout << "Settings opened!" << endl;
                break;
            case 4:
                cout << "Goodbye!" << endl;
                break;
            default:
                cout << "Invalid choice! Try again." << endl;
        }
    } while (choice != 4);
    
    return 0;
}
```

### Example 2: Input Validation
```cpp
#include <iostream>
using namespace std;

int main() {
    int age;
    
    do {
        cout << "Enter your age (0-150): ";
        cin >> age;

            cout << "Invalid age! Please try again." << endl;
        }
    } while (age < 0 || age > 150);
    
    cout << "Age accepted: " << age << endl;
    
    return 0;

```

### Example 3: Simple Calculator
```cpp
#include <iostream>
using namespace std;

int main() {
    double num1, num2, result;
    char operation;
    char continue_calc;
    
    do {
        cout << "\nEnter first number: ";
        cin >> num1;
        cout << "Enter operation (+, -, *, /): ";
        cin >> operation;
        cout << "Enter second number: ";
        cin >> num2;
        
        switch (operation) {
            case '+':
                result = num1 + num2;
                break;
            case '-':
                result = num1 - num2;
                break;
            case '*':
                result = num1 * num2;
                break;
            case '/':
                if (num2 != 0) {
                    result = num1 / num2;
                } else {
                    cout << "Cannot divide by zero!" << endl;
                    continue;
                }
                break;
            default:
                cout << "Invalid operation!" << endl;
                continue;
        }
        
        cout << num1 << " " << operation << " " << num2 << " = " << result << endl;
        
        cout << "Do another calculation? (y/n): ";
        cin >> continue_calc;
        
    } while (continue_calc == 'y' || continue_calc == 'Y');
    
    cout << "Calculator closed." << endl;
    
    return 0;
}
```

### Example 4: Number Guessing Game with Play Again
```cpp
#include <iostream>
#include <cstdlib>
#include <ctime>
using namespace std;

int main() {
    srand(time(0));
    int secret, guess, attempts;
    char playAgain;
    
    do {
        secret = rand() % 100 + 1;  // Random 1-100
        attempts = 0;
        
        cout << "\n=== GUESS THE NUMBER (1-100) ===" << endl;
        
        do {
            cout << "Enter guess: ";
            cin >> guess;
            attempts++;
            
            if (guess < secret) {
                cout << "Too low! Try again." << endl;
            } else if (guess > secret) {
                cout << "Too high! Try again." << endl;
            }
        } while (guess != secret);
        
        cout << "Correct! You got it in " << attempts << " attempts!" << endl;
        
        cout << "Play again? (y/n): ";
        cin >> playAgain;
        
    } while (playAgain == 'y' || playAgain == 'Y');
    
    cout << "Thanks for playing!" << endl;
    
    return 0;
}
```

### Example 5: ATM System
```cpp
#include <iostream>
using namespace std;

int main() {
    double balance = 1000.0;
    int choice;
    double amount;
    
    do {
        cout << "\n=== ATM SYSTEM ===" << endl;

                    cout << "Deposited: $" << amount << endl;
                    cout << "New balance: $" << balance << endl;
                } else {
                    cout << "Invalid amount!" << endl;
                }
                break;

                    cout << "Insufficient funds!" << endl;
                } else {
                    cout << "Invalid amount!" << endl;
                }
                break;
                
            case 4:
                cout << "Thank you for using ATM!" << endl;
                break;
                
            default:
                cout << "Invalid choice! Please try again." << endl;
        }
    } while (choice != 4);
    
    return 0;
}
```

## do-while for Input Validation

### Validate Range
```cpp
#include <iostream>
using namespace std;

int main() {
    int score;
    
    do {
        cout << "Enter score (0-100): ";
        cin >> score;
    } while (score < 0 || score > 100);
    
    cout << "Valid score: " << score << endl;
    
    return 0;
}
```

### Validate Specific Values
```cpp
#include <iostream>
using namespace std;

int main() {
    char grade;
    
    do {
        cout << "Enter grade (A, B, C, D, F): ";
        cin >> grade;
        grade = toupper(grade);
    } while (grade != 'A' && grade != 'B' && grade != 'C' && grade != 'D' && grade != 'F');
    
    cout << "Valid grade: " << grade << endl;
    
    return 0;
}
```

## Nested do-while Loops

```cpp
#include <iostream>
using namespace std;

int main() {
    int i = 1;
    
    do {
        int j = 1;
        
        do {
            cout << i << "," << j << "\t";
            j++;
        } while (j <= 3);
        
        cout << endl;
        i++;
    } while (i <= 3);
    
    return 0;
}
```

## Common Mistakes

### Mistake 1: Forgetting Semicolon
```cpp
int i = 1;

// Wrong - missing semicolon
do {
    cout << i << " ";
    i++;
} while (i <= 5)  // ❌ Missing semicolon!

// Correct
do {
    cout << i << " ";
    i++;
} while (i <= 5);
```

### Mistake 2: Wrong Condition Order
```cpp
int i = 1;

// Wrong - condition at top (that's while)
do {
    cout << i << " ";
    i++;
} 
while (i <= 5);  // Condition must be at bottom
```

### Mistake 3: Forgetting Braces
```cpp
int i = 1;

// Wrong - only first line is in loop
do
    cout << i << " ";  // This is in loop
    i++;                // This is outside!
while (i <= 5);

// Correct
do {
    cout << i << " ";
    i++;
} while (i <= 5);
```

### Mistake 4: Infinite Loop
```cpp
int i = 1;

// Wrong - missing increment
do {
    cout << i << " ";  // i never changes → infinite loop
} while (i <= 5);

// Correct
do {
    cout << i << " ";
    i++;  // Update counter
} while (i <= 5);
```

### Mistake 5: Using = Instead of ==
```cpp
char choice = 'y';

// Wrong - assignment instead of comparison
do {
    cout << "Looping..." << endl;
    cout << "Continue? (y/n): ";
    cin >> choice;
} while (choice = 'y');  // Always true!

// Correct
do {
    cout << "Looping..." << endl;
    cout << "Continue? (y/n): ";
    cin >> choice;
} while (choice == 'y');
```

## When to Use do-while vs while

### Use do-while when:
- Loop must execute at least once (menus, input validation)
- Condition depends on user input inside loop
- You need to show something before asking

### Use while when:
- Loop might not need to execute at all
- Condition is known before first iteration
- Reading data until EOF

## Comparison Table

| Feature | while | do-while |
|---------|-------|----------|
| Condition check | At beginning | At end |
| Minimum executions | 0 | 1 |
| Syntax | `while (c) { }` | `do { } while (c);` |
| Semicolon after | No | Yes |
| Best for | Pre-check conditions | Post-check conditions |

## Quick Reference

| Pattern | Description |
|---------|-------------|
| `do { } while (c);` | Basic do-while |
| `do { } while (i <= n);` | Counting loop |
| `do { } while (choice != 4);` | Menu loop |
| `do { } while (invalid);` | Validation loop |

## Summary

- **do-while** guarantees at least one execution
- Condition checked after loop body
- Always ends with semicolon `;`
- Perfect for menus and input validation
- Body executes first, then condition checked
- Use when you need to show options before asking
- Watch for infinite loops (update variables!)
- Similar to while but with post-condition check
- Good for "ask and repeat" patterns

# 22 - if-else Statement in C++

## What is if-else?
The `if-else` statement provides two paths: one block executes if condition is true, another block executes if condition is false.

## Basic Syntax

```cpp
if (condition) {
    // code executes if condition is true
} else {
    // code executes if condition is false
}
```

## Simple Example

```cpp
#include <iostream>
using namespace std;

int main() {
    int number = 7;
    
    if (number % 2 == 0) {
        cout << number << " is even" << endl;
    } else {
        cout << number << " is odd" << endl;
    }
    
    return 0;
}
```
Output:
```
7 is odd
```

## How if-else Works

```cpp
#include <iostream>
using namespace std;

int main() {
    int age;
    
    cout << "Enter your age: ";
    cin >> age;
    
    if (age >= 18) {
        cout << "You are an adult." << endl;
        cout << "You can vote." << endl;
    } else {
        cout << "You are a minor." << endl;
        cout << "You cannot vote yet." << endl;
    }
    
    return 0;
}
```

## Common Mistakes

### Mistake 1: Semicolon After if
```cpp
int age = 15;

// Wrong
if (age >= 18); {
    cout << "Adult" << endl;  // Always runs!
} else {
    cout << "Minor" << endl;  // Error!
}

// Correct
if (age >= 18) {
    cout << "Adult" << endl;
} else {
    cout << "Minor" << endl;
}
```

### Mistake 2: Using = Instead of ==
```cpp
int x = 5;

// Wrong
if (x = 10) {
    cout << "x is 10" << endl;  // Always runs, x becomes 10
} else {
    cout << "x is not 10" << endl;
}

// Correct
if (x == 10) {
    cout << "x is 10" << endl;
} else {
    cout << "x is not 10" << endl;
}
```

### Mistake 3: Missing Braces
```cpp
int score = 75;

// Wrong - only first line is in if/else
if (score >= 60)
    cout << "Passed" << endl;
    cout << "Congratulations" << endl;  // Always runs
else
    cout << "Failed" << endl;

// Correct
if (score >= 60) {
    cout << "Passed" << endl;
    cout << "Congratulations" << endl;
} else {
    cout << "Failed" << endl;
}
```

### Mistake 4: Else Without If
```cpp
// Wrong
else {
    cout << "This won't work" << endl;
}

// Correct
if (condition) {
    // something
} else {
    cout << "This works" << endl;
}
```

### Mistake 5: Wrong Condition Logic
```cpp
int x = 5;

// Wrong
if (x = 5) {  // Assignment, not comparison
    cout << "x is 5" << endl;
} else {
    cout << "x is not 5" << endl;
}

// Correct
if (x == 5) {
    cout << "x is 5" << endl;
} else {
    cout << "x is not 5" << endl;
}
```

## if-else with Multiple Conditions

```cpp
#include <iostream>
using namespace std;

int main() {
    int age = 25;
    bool hasTicket = true;
    
    if (age >= 18 && hasTicket) {
        cout << "Welcome to the show!" << endl;
    } else {
        cout << "Cannot enter." << endl;
        
        if (age < 18) {
            cout << "Too young." << endl;
        }
        if (!hasTicket) {
            cout << "No ticket." << endl;
        }
    }
    
    return 0;
}
```

## Quick Reference

| Code | Meaning |
|------|---------|
| `if (condition) { }` | If true, do this |
| `if (condition) { } else { }` | If true do first, else do second |
| `if (x > 5) { } else { }` | Check if x greater than 5 |
| `if (isValid) { } else { }` | Check boolean directly |

## Summary

- **if-else** provides two execution paths
- **if block** runs when condition is true
- **else block** runs when condition is false
- Always use braces `{}` for clarity
- No condition after `else`
- Can nest if-else inside blocks
- Use `==` for comparison, not `=`
- Test both true and false cases
- Indent code properly for readability

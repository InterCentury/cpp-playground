# 26 - Ternary Operator (?:) in C++

## What is the Ternary Operator?
The ternary operator is a shorthand way of writing an if-else statement. It's called "ternary" because it takes three operands.

## Basic Syntax

```cpp
condition ? expression1 : expression2;
```

- If **condition** is true, **expression1** is executed/returned
- If **condition** is false, **expression2** is executed/returned

## Simple Example

```cpp
#include <iostream>
using namespace std;

int main() {
    int age = 20;
    string result;
    
    // Using ternary operator
    result = (age >= 18) ? "Adult" : "Minor";
    
    cout << "Status: " << result << endl;
    
    return 0;
}
```
Output:
```
Status: Adult
```

## Ternary vs if-else

```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 10, y = 5;
    int max;
    
    // Using if-else
    if (x > y) {
        max = x;
    } else {
        max = y;
    }
    cout << "Max (if-else): " << max << endl;
    
    // Using ternary (shorter)
    max = (x > y) ? x : y;
    cout << "Max (ternary): " << max << endl;
    
    return 0;
}
```

## Using Ternary for Assignment

```cpp
#include <iostream>
using namespace std;

int main() {
    int score = 75;
    string grade;
    double bonus = 0;
    
    // Assign based on condition
    grade = (score >= 60) ? "Pass" : "Fail";
    cout << "Result: " << grade << endl;
    
    // Numeric assignment
    bonus = (score >= 90) ? 100 : (score >= 75) ? 50 : 0;
    cout << "Bonus: $" << bonus << endl;
    
    return 0;
}
```

## Ternary for Output

```cpp
#include <iostream>
using namespace std;

int main() {
    int number = 7;
    
    // Directly in cout
    cout << number << " is " << ((number % 2 == 0) ? "even" : "odd") << endl;
    
    int temperature = 25;
    cout << "Weather: " << (temperature > 30 ? "Hot" : "Nice") << endl;
    
    return 0;
}
```
Output:
```
7 is odd
Weather: Nice
```

## Practical Examples

### Example 1: Find Maximum of Two Numbers
```cpp
#include <iostream>
using namespace std;

int main() {
    int a, b;
    
    cout << "Enter two numbers: ";
    cin >> a >> b;
    
    int max = (a > b) ? a : b;
    int min = (a < b) ? a : b;
    
    cout << "Maximum: " << max << endl;
    cout << "Minimum: " << min << endl;
    
    return 0;
}
```

### Example 2: Even or Odd
```cpp
#include <iostream>
using namespace std;

int main() {
    int num;
    
    cout << "Enter a number: ";
    cin >> num;
    
    cout << num << " is " << ((num % 2 == 0) ? "even" : "odd") << endl;
    
    return 0;
}
```

### Example 3: Absolute Value
```cpp
#include <iostream>
using namespace std;

int main() {
    int num;
    
    cout << "Enter a number (can be negative): ";
    cin >> num;
    
    int absolute = (num < 0) ? -num : num;
    
    cout << "Absolute value: " << absolute << endl;
    
    return 0;
}
```

### Example 4: Pass/Fail Checker
```cpp
#include <iostream>
using namespace std;

int main() {
    int score;
    
    cout << "Enter test score: ";
    cin >> score;
    
    string result = (score >= 60) ? "PASSED" : "FAILED";
    char grade = (score >= 90) ? 'A' : 
                 (score >= 80) ? 'B' : 
                 (score >= 70) ? 'C' : 
                 (score >= 60) ? 'D' : 'F';
    
    cout << "Result: " << result << endl;
    cout << "Grade: " << grade << endl;
    
    return 0;
}
```

### Example 5: Simple Calculator
```cpp
#include <iostream>
using namespace std;

int main() {
    double a, b;
    char op;
    double result;
    
    cout << "Enter expression (e.g., 5 + 3): ";
    cin >> a >> op >> b;
    
    result = (op == '+') ? a + b :
             (op == '-') ? a - b :
             (op == '*') ? a * b :
             (op == '/' && b != 0) ? a / b : 0;
    
    if (op == '/' && b == 0) {
        cout << "Error: Division by zero!" << endl;
    } else {
        cout << a << " " << op << " " << b << " = " << result << endl;
    }
    
    return 0;
}
```

## Nested Ternary Operators

```cpp
#include <iostream>
using namespace std;

int main() {
    int num = 0;
    
    // Nested ternary to check positive/negative/zero
    string result = (num > 0) ? "positive" : 
                    (num < 0) ? "negative" : 
                    "zero";
    
    cout << num << " is " << result << endl;
    
    // Grade with nested ternary
    int score = 85;
    char grade = (score >= 90) ? 'A' :
                 (score >= 80) ? 'B' :
                 (score >= 70) ? 'C' :
                 (score >= 60) ? 'D' : 'F';
    
    cout << "Score: " << score << ", Grade: " << grade << endl;
    
    return 0;
}
```

## Ternary with Multiple Statements

```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 5, y = 3;
    
    // You can call functions
    (x > y) ? cout << "x is greater" << endl : cout << "y is greater" << endl;
    
    // But can't have multiple statements without comma operator
    (x > y) ? (cout << "x > y", cout << " x wins") : (cout << "y > x", cout << " y wins");
    cout << endl;
    
    return 0;
}
```

## Using Ternary in Function Arguments

```cpp
#include <iostream>
using namespace std;

void printMessage(string msg, bool urgent) {
    cout << (urgent ? "URGENT: " : "Info: ") << msg << endl;
}

int main() {
    printMessage("Server is running", false);
    printMessage("Low disk space", true);
    
    return 0;
}
```

## Ternary with Different Types

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 20;
    
    // Both expressions must be compatible types
    double result = (a > b) ? 3.14 : 2.5;      // OK - both numeric
    string text = (a > b) ? "greater" : "less"; // OK - both strings
    
    // This would cause issues:
    // auto bad = (a > b) ? 10 : "hello";  // Different types!
    
    cout << "Result: " << result << endl;
    cout << "Text: " << text << endl;
    
    return 0;
}
```

## Common Mistakes

### Mistake 1: Forgetting Parentheses
```cpp
int x = 5, y = 3;

// Wrong - << has higher precedence than ?
cout << "Max is " << x > y ? x : y << endl;  // Error!

// Correct
cout << "Max is " << (x > y ? x : y) << endl;
```

### Mistake 2: Using Statements Instead of Expressions
```cpp
int x = 5;

// Wrong - can't put statements
x > 0 ? cout << "Positive" : return 0;  // Error!

// Correct
if (x > 0) {
    cout << "Positive";
} else {
    return 0;
}
```

### Mistake 3: Type Mismatch
```cpp
int a = 10;

// Wrong - different types
string result = (a > 5) ? "yes" : 123;  // Error!

// Correct
string result = (a > 5) ? "yes" : "123";
```

### Mistake 4: Overcomplicating
```cpp
// Hard to read
int result = (x > y) ? ((x > z) ? x : z) : ((y > z) ? y : z);

// Better with function
int max3(int a, int b, int c) {
    return (a > b) ? ((a > c) ? a : c) : ((b > c) ? b : c);
}
```

### Mistake 5: Missing Lvalue
```cpp
int x = 5;

// Wrong - ternary returns value, not lvalue
(x > 0 ? x : 0) = 10;  // Error!

// Correct
if (x > 0) {
    x = 10;
}
```

## When to Use Ternary vs if-else

### Use Ternary For:
```cpp
// Simple assignments
int max = (a > b) ? a : b;

// Quick conditions in cout
cout << (isValid ? "Valid" : "Invalid");

// Return values
return (num > 0) ? num : -num;
```

### Use if-else For:
```cpp
// Complex logic
if (user.isAdmin() && user.hasAccess() || emergencyMode) {
    grantFullAccess();
} else {
    grantLimitedAccess();
}

// Multiple statements
if (condition) {
    doThis();
    doThat();
    updateStatus();
} else {
    handleFailure();
}

// Long conditions
if (score >= 90 && attendance >= 80 && projectsCompleted >= 5) {
    grade = 'A';
}
```

## Performance Comparison

```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 10, y = 5, result;
    
    // These are equivalent in performance
    // Compiler optimizes both
    
    // if-else
    if (x > y) {
        result = x;
    } else {
        result = y;
    }
    
    // ternary
    result = (x > y) ? x : y;
    
    return 0;
}
```

## Quick Reference

| Expression | Meaning |
|------------|---------|
| `c ? t : f` | If c true, evaluate t, else evaluate f |
| `(a > b) ? a : b` | Maximum of a and b |
| `(num >= 0) ? num : -num` | Absolute value |
| `(score >= 60) ? "Pass" : "Fail"` | Pass/fail message |
| `(x % 2 == 0) ? "even" : "odd"` | Even/odd check |

## Summary

- **Ternary operator** is shorthand for if-else
- Syntax: `condition ? expr1 : expr2`
- Returns value of expr1 if true, expr2 if false
- Use for simple conditional assignments
- Can be nested but may become hard to read
- Both expressions must be compatible types
- Great for cout statements
- Not suitable for complex logic or multiple statements
- More concise than if-else for simple conditions
- Parentheses often needed for precedence

## Basic Template

#include <iostream>

}

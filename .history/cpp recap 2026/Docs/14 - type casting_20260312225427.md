# 14 - Type Casting in C++

## What is Type Casting?
Type casting is converting a value from one data type to another. For example, converting an `int` to a `float`, or a `double` to an `int`.

## Why Type Casting?

```cpp
int a = 5;
int b = 2;
float result = a / b;     // result = 2.0 (not 2.5!)
// Because both a and b are int, division gives int

// Fix with type casting
float result = (float)a / b;  // result = 2.5
```

## Two Types of Casting

| Type | Description | Safety |
|------|-------------|--------|
| **Implicit** | Automatic conversion by compiler | Generally safe |
| **Explicit** | Manual conversion by programmer | You control it |

## 1. Implicit Type Conversion (Automatic)

```cpp
#include <iostream>
using namespace std;

int main() {
    // int to float
    int x = 10;
    float y = x;  // Automatically converts 10 to 10.0f
    cout << "int to float: " << y << endl;
    
    // char to int
    char ch = 'A';
    int ascii = ch;  // Automatically converts 'A' to 65
    cout << "char to int: " << ascii << endl;
    
    // float to double
    float f = 3.14f;
    double d = f;    // Automatically converts
    cout << "float to double: " << d << endl;
    
    // bool to int
    bool b = true;
    int i = b;       // true becomes 1
    cout << "bool to int: " << i << endl;
    
    return 0;
}
```

### Implicit Conversion Hierarchy
```
bool → char → short → int → unsigned int → long → unsigned long → 
long long → float → double → long double
```
(Lower types automatically convert to higher types)

## 2. Explicit Type Casting (Manual)

### Method 1: C-style Cast (Old Way)
```cpp
#include <iostream>
using namespace std;

int main() {
    double pi = 3.14159;
    
    int intPi = (int)pi;           // C-style cast
    float floatPi = (float)pi;      // C-style cast
    
    cout << "Original pi: " << pi << endl;
    cout << "int pi: " << intPi << endl;      // 3 (truncated)
    cout << "float pi: " << floatPi << endl;  // 3.14159
    
    return 0;
}
```

### Method 2: Function-style Cast
```cpp
int a = 10;
double b = double(a);      // Function-style cast
float c = float(a);
```

### Method 3: C++ Cast Operators (Modern Way)

C++ provides four specific cast operators for different situations:

## The Four C++ Casts

### 1. static_cast (Most Common)
For standard conversions between related types.

```cpp
#include <iostream>
using namespace std;

int main() {
    double pi = 3.14159;
    
    // Safe, compile-time checked
    int intPi = static_cast<int>(pi);
    float floatPi = static_cast<float>(pi);
    
    cout << "static_cast<int> pi: " << intPi << endl;
    
    // char to int
    char ch = 'A';
    int ascii = static_cast<int>(ch);
    cout << "static_cast<int> 'A': " << ascii << endl;
    
    // int to char
    int num = 66;
    char letter = static_cast<char>(num);
    cout << "static_cast<char> 66: " << letter << endl;  // 'B'
    
    return 0;
}
```

### 2. dynamic_cast
For polymorphic class hierarchies (used with inheritance).

```cpp
class Base { virtual void dummy() {} };
class Derived : public Base {};

Base* basePtr = new Derived();
Derived* derivedPtr = dynamic_cast<Derived*>(basePtr);  // Safe downcast
```

### 3. const_cast
To add or remove const qualifier.

```cpp
#include <iostream>
using namespace std;

void print(char* str) {
    cout << str << endl;
}

int main() {
    const char* message = "Hello";
    // print(message);  // ❌ Error! const char* to char*
    
    // Remove const temporarily
    print(const_cast<char*>(message));  // ✅ OK
    
    return 0;
}
```

### 4. reinterpret_cast
For low-level reinterpreting of bit patterns (dangerous!).

```cpp
#include <iostream>
using namespace std;

int main() {
    int num = 65;
    
    // Treat int as pointer (rarely needed)
    char* ptr = reinterpret_cast<char*>(&num);
    cout << "As char: " << *ptr << endl;  // Might print 'A'
    
    return 0;
}
```

## Common Casting Examples

### Example 1: Integer Division Fix
```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 7, b = 2;
    
    // Without casting
    int result1 = a / b;
    cout << "Without cast: " << result1 << endl;  // 3
    
    // With C-style cast
    float result2 = (float)a / b;
    cout << "C-style cast: " << result2 << endl;  // 3.5
    
    // With static_cast
    float result3 = static_cast<float>(a) / b;
    cout << "static_cast: " << result3 << endl;   // 3.5
    
    return 0;
}
```

### Example 2: Char to Int Conversions
```cpp
#include <iostream>
using namespace std;

int main() {
    // Convert digit char to number
    char digit = '7';
    int num = digit - '0';  // Common trick
    cout << "Digit char to int: " << num << endl;
    
    // Using static_cast
    char ch = '9';
    int val = static_cast<int>(ch - '0');
    cout << "static_cast digit: " << val << endl;
    
    // Convert number to digit char
    int number = 5;
    char digitChar = '0' + number;
    cout << "Number to digit char: " << digitChar << endl;
    
    return 0;
}
```

### Example 3: Precision Control
```cpp
#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    double precise = 123.456789;
    
    // Truncate to integer
    int truncated = static_cast<int>(precise);
    cout << "Truncated: " << truncated << endl;  // 123
    
    // Round by adding 0.5 then cast
    int rounded = static_cast<int>(precise + 0.5);
    cout << "Rounded: " << rounded << endl;      // 123 (123.456+0.5=123.956 → 123)
    
    // For 123.567
    double test = 123.567;
    rounded = static_cast<int>(test + 0.5);
    cout << "Rounded 123.567: " << rounded << endl;  // 124
    
    return 0;
}
```

### Example 4: Safe Numeric Conversions
```cpp
#include <iostream>
using namespace std;

int main() {
    // Potential overflow
    int maxInt = 2147483647;
    
    // Safe conversion to larger type
    long long larger = static_cast<long long>(maxInt) * 2;
    cout << "maxInt * 2 (safe): " << larger << endl;
    
    // Without cast - overflow
    int overflow = maxInt * 2;  // Undefined behavior
    cout << "maxInt * 2 (overflow): " << overflow << endl;
    
    return 0;
}
```

### Example 5: Mixed Type Calculations
```cpp
#include <iostream>
using namespace std;

int main() {
    int apples = 10;
    double pricePerApple = 1.99;
    
    // Without cast - int * double = double (implicit)
    double total1 = apples * pricePerApple;
    cout << "Total (implicit): $" << total1 << endl;
    
    // Explicit cast for clarity
    double total2 = static_cast<double>(apples) * pricePerApple;
    cout << "Total (explicit): $" << total2 << endl;
    
    // Average calculation
    int scores[] = {85, 92, 78, 90, 88};
    int sum = 0;
    for(int i = 0; i < 5; i++) {
        sum += scores[i];
    }
    
    // Without cast - integer division
    int avg1 = sum / 5;
    cout << "Average (int): " << avg1 << endl;
    
    // With cast - floating point division
    double avg2 = static_cast<double>(sum) / 5;
    cout << "Average (double): " << avg2 << endl;
    
    return 0;
}
```

## Casting Between Different Types

### Numeric to Numeric
```cpp
int i = 42;
double d = static_cast<double>(i);     // int → double
float f = static_cast<float>(i);       // int → float
short s = static_cast<short>(i);       // int → short
```

### Numeric to Boolean
```cpp
int i = 10;
bool b = static_cast<bool>(i);  // true (non-zero)
i = 0;
b = static_cast<bool>(i);       // false
```

### Pointer to Pointer
```cpp
int* intPtr = new int(42);
void* voidPtr = static_cast<void*>(intPtr);  // int* → void*
// intPtr = static_cast<int*>(voidPtr);       // void* → int*
```

## Dangerous Casts to Avoid

### 1. Casting Away const (without need)
```cpp
const int value = 42;
int* ptr = const_cast<int*>(&value);
*ptr = 100;  // Undefined behavior!
```

### 2. reinterpret_cast for simple conversions
```cpp
float f = 3.14f;
int i = reinterpret_cast<int&>(f);  // Treats float bits as int
// Use static_cast if possible: int i = static_cast<int>(f);
```

### 3. Downcasting without dynamic_cast
```cpp
Base* base = new Base();
Derived* derived = static_cast<Derived*>(base);  // Dangerous!
```

## Type Casting Table

| From | To | Implicit? | Cast Needed | Example |
|------|-----|-----------|-------------|---------|
| `int` | `float` | ✅ Yes | Optional | `float f = i;` |
| `float` | `int` | ❌ No | Required | `int i = (int)f;` |
| `char` | `int` | ✅ Yes | Optional | `int i = ch;` |
| `int` | `char` | ❌ No | Required | `char c = (char)i;` |
| `double` | `float` | ❌ No | Required | `float f = (float)d;` |
| `int*` | `void*` | ✅ Yes | Optional | `void* v = ptr;` |
| `void*` | `int*` | ❌ No | Required | `int* p = (int*)v;` |

## Common Mistakes

### Mistake 1: Losing Precision
```cpp
double precise = 3.1415926535;
float approx = static_cast<float>(precise);  // Precision lost!
cout << fixed << setprecision(10);
cout << "Original: " << precise << endl;  // 3.1415926535
cout << "After cast: " << approx << endl;  // 3.1415927410
```

### Mistake 2: Integer Division
```cpp
// Wrong
int x = 7, y = 2;
float result = x / y;  // result = 3.0, not 3.5!

// Correct
float result = static_cast<float>(x) / y;
```

### Mistake 3: Overflow
```cpp
int large = 1000000;
char c = static_cast<char>(large);  // Overflow! Data loss
```

### Mistake 4: Casting Between Unrelated Types
```cpp
// Wrong - nonsensical
string str = "Hello";
int* ptr = reinterpret_cast<int*>(&str);  // Don't do this!
```

### Mistake 5: Forgetting Cast in Calculations
```cpp
int total = 95;
int count = 4;
double average = total / count;  // average = 23.0, not 23.75!
```

## Best Practices

### ✅ Do This
```cpp
// Use static_cast for most conversions
double d = 3.14;
int i = static_cast<int>(d);

// Cast before division
double avg = static_cast<double>(sum) / count;

// Use digit - '0' for char to int
char digit = '5';
int num = digit - '0';

// Use '0' + num for int to char
int value = 7;
char ch = '0' + value;

// Check for overflow when casting down
long long big = 1000000;
if (big <= INT_MAX) {
    int safe = static_cast<int>(big);
}
```

### ❌ Avoid This
```cpp
// Don't use C-style casts in C++
int i = (int)3.14;  // OK but prefer static_cast

// Don't use reinterpret_cast for simple conversions
float f = 3.14f;
int bits = reinterpret_cast<int&>(f);  // Too dangerous

// Don't cast away const unnecessarily
const int x = 10;
int& y = const_cast<int&>(x);  // Avoid!

// Don't cast between unrelated types
string s = "test";
int* p = (int*)&s;  // Wrong!
```

## Quick Reference

| Cast Type | Syntax | Use For |
|-----------|--------|---------|
| C-style | `(type)value` | Quick, old code (avoid in C++) |
| Function-style | `type(value)` | Looks like constructor |
| static_cast | `static_cast<type>(value)` | **Most conversions** (safe, checked) |
| dynamic_cast | `dynamic_cast<type*>(ptr)` | Polymorphic class hierarchies |
| const_cast | `const_cast<type>(value)` | Adding/removing const |
| reinterpret_cast | `reinterpret_cast<type>(value)` | Low-level bit reinterpretation (rare) |

## Common Conversion Patterns

```cpp
// String to number
string numStr = "123";
int num = stoi(numStr);          // String to int
double dnum = stod(numStr);       // String to double

// Number to string
int value = 42;
string str = to_string(value);    // int to string

// Char to int (digit)
char ch = '7';
int digit = ch - '0';             // '7' → 7

// Int to char (digit)
int digit = 5;
char ch = '0' + digit;            // 5 → '5'

// Char to int (ASCII)
char letter = 'A';
int ascii = static_cast<int>(letter);  // 65
```

## Summary

- **Type casting** converts values between different types
- **Implicit conversion** happens automatically (usually safe)
- **Explicit conversion** you control with casts
- **static_cast** is the modern, safe choice for most conversions
- Be careful with **narrowing conversions** (double → int)
- Integer division needs casting for decimal results
- Use digit - '0' for char to int conversions
- Casting doesn't change original variable, creates new value
- Four C++ casts: static_cast, dynamic_cast, const_cast, reinterpret_cast
- Prefer C++ casts over C-style casts

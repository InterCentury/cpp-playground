# 04 - cout for Output in C++

## What is cout?
`cout` (pronounced "see-out") is used to display output on the screen. It stands for "character output".

## Basic Syntax
```cpp
cout << "text here";
```
- `<<` is the insertion operator (sends data to cout)

## Simple Examples

### Example 1: Printing Text
```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello World";
    return 0;
}
```
Output:
```
Hello World
```

### Example 2: Printing Multiple Items
```cpp
cout << "Hello" << " " << "World";
```
Output:
```
Hello World
```

## Printing Different Data Types

### Numbers
```cpp
cout << 42;        // Integer
cout << 3.14;      // Decimal
cout << 10 + 5;    // Expression result
```
Output:
```
423.1415
```

### Variables
```cpp
int age = 25;
double price = 99.99;
char grade = 'A';

cout << age;       // 25
cout << price;     // 99.99
cout << grade;     // A
```

### Mixing Text and Variables
```cpp
string name = "John";
int age = 25;

cout << "Name: " << name << ", Age: " << age;
```
Output:
```
Name: John, Age: 25
```

## Adding New Lines

### Method 1: Using `endl`
```cpp
cout << "First line" << endl;
cout << "Second line" << endl;
```

### Method 2: Using `\n`
```cpp
cout << "First line\n";
cout << "Second line\n";
```

Output for both:
```
First line
Second line
```

## Formatting Output

### Multiple Lines in One cout
```cpp
cout << "Line 1\nLine 2\nLine 3";
```

### Adding Spaces
```cpp
cout << "Hello" << "    " << "World";
// Hello    World
```

## Common Examples

### Example 1: Simple Greeting
```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Welcome to C++!" << endl;
    cout << "Let's learn together.";
    return 0;
}
```

### Example 2: Displaying Calculations
```cpp
int x = 10, y = 5;
cout << "Sum = " << x + y << endl;
cout << "Product = " << x * y << endl;
```
Output:
```
Sum = 15
Product = 50
```

### Example 3: Table Format
```cpp
cout << "Name\tAge\tCity" << endl;
cout << "John\t25\tNYC" << endl;
cout << "Anna\t30\tLA" << endl;
```
Output:
```
Name    Age    City
John    25     NYC
Anna    30     LA
```
(`\t` adds a tab space)

## cout with Calculations

### Direct Calculations
```cpp
cout << 5 + 3 << endl;        // 8
cout << 10 - 4 << endl;        // 6
cout << 6 * 7 << endl;         // 42
cout << 20 / 5 << endl;        // 4
```

### Using Variables
```cpp
int a = 15, b = 4;
cout << a + b << endl;    // 19
cout << a - b << endl;    // 11
cout << a * b << endl;    // 60
cout << a / b << endl;    // 3 (integer division)
```

## Common Patterns

### Pattern 1: Print and End Line
```cpp
cout << "Hello" << endl;
cout << "World" << endl;
```

### Pattern 2: Build Output Gradually
```cpp
cout << "Result: ";
cout << 42;
cout << endl;
// Result: 42
```

### Pattern 3: Chain Everything
```cpp
cout << "Value = " << 100 << " and " << 200 << endl;
// Value = 100 and 200
```

## Quick Reference Table

| What to Print | How to Write | Example |
|--------------|--------------|---------|
| Text | `"text"` | `cout << "Hi";` |
| Number | just number | `cout << 25;` |
| Variable | variable name | `cout << age;` |
| New line | `endl` or `\n` | `cout << endl;` |
| Tab | `\t` | `cout << "\t";` |
| Mix | use `<<` between | `cout << a << b;` |

## Common Errors

### Error 1: Missing `<<`
```cpp
// Wrong
cout "Hello";

// Correct
cout << "Hello";
```

### Error 2: Missing Quotes for Text
```cpp
// Wrong
cout << Hello;

// Correct
cout << "Hello";
```

### Error 3: Missing Semicolon
```cpp
// Wrong
cout << "Hello"

// Correct
cout << "Hello";
```

### Error 4: Wrong Quotes for Characters
```cpp
// Wrong (use " for text)
cout << 'Hello';

// Correct
cout << "Hello";
```

## Practice Examples

### Example 1: Personal Info
```cpp
string name = "Alex";
int age = 22;
string city = "Boston";

cout << "Name: " << name << endl;
cout << "Age: " << age << endl;
cout << "City: " << city << endl;
```

### Example 2: Shopping Receipt
```cpp
string item = "Laptop";
double price = 999.99;
int quantity = 2;
double total = price * quantity;

cout << "Item: " << item << endl;
cout << "Price: $" << price << endl;
cout << "Quantity: " << quantity << endl;
cout << "Total: $" << total << endl;
```

### Example 3: Simple Calculator Display
```cpp
int num1 = 12, num2 = 4;

cout << num1 << " + " << num2 << " = " << num1 + num2 << endl;
cout << num1 << " - " << num2 << " = " << num1 - num2 << endl;
cout << num1 << " × " << num2 << " = " << num1 * num2 << endl;
cout << num1 << " ÷ " << num2 << " = " << num1 / num2 << endl;
```

## Escape Sequences

| Sequence | Meaning | Example Output |
|----------|---------|----------------|
| `\n` | New line | `"Hi\nBye"` → Hi (new line) Bye |
| `\t` | Tab | `"A\tB"` → A    B |
| `\"` | Double quote | `"Say \"Hi\""` → Say "Hi" |
| `\\` | Backslash | `"Path\\file"` → Path\file |

## Summary

- `cout` prints to screen using `<<` operator
- Text must be in `"quotes"`, numbers don't need quotes
- Use `endl` or `\n` to add new lines
- Multiple items can be chained with `<<`
- `\t` adds tab space for formatting
- Calculations can be done directly inside cout
- Every cout statement ends with `;`

## Basic Template
```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Your message here" << endl;
    return 0;
}
```
